🔵 파이썬 마스터 코치: 18. 리스트 응용

리스트를 자유자재로 다루는 고수의 기술을 익혀봅시다!

1️⃣ enumerate(): 번호표 붙이기 🎫

리스트의 데이터(값)뿐만 아니라 순서(인덱스)도 필요할 때 사용합니다.

🎫 마스터 코치의 비유: 번호표
그냥 "사과, 바나나"라고 부르지 않고,
"1번 손님 사과, 2번 손님 바나나"처럼 번호표를 같이 줍니다. 
fruits = ['사과', '바나나', '포도']

# 일반 for문: 값만 나옴
for fruit in fruits:
    print(fruit)

# enumerate 사용: (번호, 값) 쌍으로 나옴
for i, fruit in enumerate(fruits):
    print(i + 1, "번 과일:", fruit)

# 출력 결과:
# 2 번 과일: 바나나...
🤔 궁금증: for문 없이 그냥 출력하면?

enumerate는 바로 리스트로 나오지 않고, "준비된 상태"(객체)로만 존재합니다. 내용을 눈으로 확인하려면 list()로 변환해야 합니다.

print(enumerate(fruits))
# 출력: <enumerate object at 0x...> (알 수 없는 기호)

print(list(enumerate(fruits)))
# 출력: [(0, '사과'), (1, '바나나'), (2, '포도')]

2️⃣ zip(): 지퍼처럼 합치기 🤐

두 개 이상의 리스트를 같은 순서끼리 짝지어서 가져옵니다.

🤐 마스터 코치의 비유: 지퍼(Zipper)
왼쪽 이빨과 오른쪽 이빨이 하나씩 맞물려 잠기듯이,
두 리스트의 아이템을 하나씩 묶어서(튜플) 줍니다. 
names = ['철수', '영희', '민수']
scores = [90, 85, 100]

# 두 리스트를 동시에 반복!
for name, score in zip(names, scores):
    print(name, "의 점수는", score, "점")

# 출력 결과:
# 영희 의 점수는 85 점...
🤔 궁금증: for문 없이 그냥 출력하면?

zip도 마찬가지로 "지퍼가 채워지기 전" 상태입니다. 결과를 보려면 list()를 사용하세요.

print(zip(names, scores))
# 출력: <zip object at 0x...>

print(list(zip(names, scores)))
# 출력: [('철수', 90), ('영희', 85), ('민수', 100)]

3️⃣ 큰 수, 작은 수 구하기 (3가지 방법) 👑

가장 큰 값(최대값)과 가장 작은 값(최소값)을 찾는 방법들입니다.

방법 1: 반복문 사용 (도장 깨기) - 원리 이해!

잠정적인 챔피언을 정해두고, 더 센 녀석이 나타나면 교체합니다.

nums = [10, 50, 30, 20]
max_val = nums[0] # 첫 번째를 임시 1등으로 설정

for n in nums:
    if n > max_val:  # 더 큰 수가 나타나면?
        max_val = n  # 1등 교체!

print("반복문 최대값:", max_val)

방법 2: sort() 사용 (줄 세우기)

순서대로 정렬한 뒤 맨 앞(최소)이나 맨 뒤(최대)를 뽑습니다.

nums.sort()
print("정렬 후 최소값:", nums[0])
print("정렬 후 최대값:", nums[-1])

방법 3: min(), max() 함수 (마법!) 🪄

파이썬이 제공하는 함수를 쓰면 한 방에 해결됩니다. (가장 추천!)

print("함수 최대값:", max(nums))
print("함수 최소값:", min(nums))

4️⃣ 합계 구하기 (2가지 방법) ➕

방법 1: 반복문 누적 (저금통 🐖)

변수(저금통)를 0으로 만들고, 숫자를 하나씩 더해나갑니다.

total = 0
for n in nums:
    total = total + n  # 기존 금액에 추가

print("반복문 합계:", total)

방법 2: sum() 함수 (계산기 🧮)

print("함수 합계:", sum(nums))

5️⃣ 📝 고수 인증 테스트

  • a. (값, 값)
  • b. (인덱스, 값)
  • c. (인덱스, 인덱스)
  • d. (길이, 값)
2. [코딩하기] 평균 점수 구하기

sum()과 len() 함수를 활용하여 리스트의 평균을 구하세요.

scores = [80, 90, 100]
total = _______(scores)  # 합계
count = _______(scores)  # 개수
average = total / count
print("평균:", average)
🔑 정답 확인하기 (클릭)
  1. b: (순서 번호, 실제 데이터) 순서로 줍니다.
  2. 코딩 2: sum, len (평균 = 합계 / 개수)

📌 핵심 요약

  • enumerate(): 순서(index)와 값(value)을 동시에!
  • zip(): 여러 리스트를 동시에!
  • max() / min() / sum(): 반복문 없이도 최대/최소/합계를 한 방에!
