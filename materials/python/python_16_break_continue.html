🔴 파이썬 마스터 코치: 16. break & continue

반복문을 내 마음대로 조종하는 리모컨을 쥐어봅시다!

1️⃣ break: 멈춰! 비상 탈출 🛑

반복문이 아직 끝나지 않았어도, break를 만나면 즉시 반복을 중단하고 탈출합니다.

🛑 마스터 코치의 비유: 자동차 브레이크
목적지에 도착하면(조건 만족), 시동을 끄고 내려야겠죠?
계속 달릴 필요 없이 "여기서 끝!" 하고 멈추는 것입니다. 
# 1부터 100까지 숫자를 찾다가...
for i in range(1, 101):
    print(i, "번째 숫자 확인 중...")
    
    if i == 5:
        print("찾았다! 여기서 끝.")
        break  # 5를 찾으면 반복문 즉시 종료!

print("반복문 끝남")

# 출력 결과: 1, 2, 3, 4, 5번까지만 출력되고 종료됨

🖐 while문 사용 예시

무한 루프(while True)를 만들고, 특정 조건에서 break로 탈출하는 패턴을 많이 씁니다.

while True:
    text = input("종료하려면 'q'를 입력하세요: ")
    
    if text == 'q':
        print("프로그램을 종료합니다.")
        break
    
    print(text, "라고 입력하셨군요.")

2️⃣ continue: 패스! 이번만 건너뛰기 🏃

반복을 멈추지는 않고, 이번 순서만 생략하고 바로 다음 순서로 넘어갑니다.

🏃 마스터 코치의 비유: 징검다리
가다가 흔들리는 돌(원치 않는 조건)이 있으면?
밟지 않고 "폴짝" 건너뛰어(Skip) 다음 돌을 밟습니다.
(길을 멈추는 게 아니라 계속 갑니다!) 
# 짝수만 출력하기 (홀수는 건너뜀)
for i in range(1, 11):
    if i % 2 == 1:    # 홀수라면?
        continue      # 밑에 코드는 무시하고 다음 숫자로 점프!
    
    print(i)          # 짝수일 때만 실행됨

# 출력 결과: 2, 4, 6, 8, 10

🖐 while문 사용 예시

while문에서 continue를 쓸 때는 조건 변경(증감) 코드의 위치에 주의해야 합니다! (무한 루프 위험)

count = 0
while count < 10:
    count = count + 1  # 1. 먼저 숫자를 늘리고
    
    if count % 2 == 1: # 2. 홀수라면?
        continue       # 3. 건너뛰기! (아래 print 실행 안 함)
        
    print(count)       # 4. 짝수만 출력

3️⃣ 📝 조종 능력 테스트

1. break와 continue의 차이점은?
  • a. 둘 다 똑같다.
  • b. break는 멈추고 나가고, continue는 이번만 건너뛴다.
  • c. break는 건너뛰고, continue는 멈춘다.
  • d. break는 파괴한다.
2. [코딩하기] 커피 자판기 (재고 관리)

커피가 5잔 있습니다. 한 잔 팔 때마다 재고를 줄이고, 0잔이 되면 "매진"을 출력하고 영업을 종료(break)하세요.

coffee = 5

while True:
    print("커피를 팝니다.")
    coffee = coffee - 1
    
    if coffee == 0:
        print("매진되었습니다.")
        ____ # 여기서 멈춰야 함!
3. [코딩하기] 3, 6, 9 게임 (박수 치기)

1부터 10까지 숫자를 출력하되, 3의 배수(3, 6, 9)에서는 숫자 대신 "짝!"을 출력하고
숫자는 출력하지 마세요. (continue 활용 가능)

for i in range(1, 11):
    if i % 3 == 0:     # 3의 배수라면
        print("짝!")
        ________       # 숫자 출력 생략하고 다음으로
    
    print(i) # 숫자 출력
🔑 정답 확인하기 (클릭)
  1. b: break는 완전 종료, continue는 이번 회차만 패스!
  2. 코딩 2: break (반복문을 완전히 빠져나옴)
  3. 코딩 3: continue (밑에 있는 print(i)를 실행하지 않고 위로 올라감)

📌 핵심 요약

  • break: 반복문을 그 즉시 탈출! (종료)
  • continue: 이번 차례만 건너뛰기! (다음 반복으로)
  • 이 두 가지를 잘 쓰면 무한 루프도 제어하고, 필요한 데이터만 쏙쏙 골라낼 수 있습니다.
