🟣 파이썬 마스터 코치: 7. 문자열 인덱싱 & 슬라이싱

글자들의 기차에서 원하는 칸을 찾아내거나, 케이크처럼 잘라볼까요?

1️⃣ 인덱싱 (Indexing): 콕 집어 가져오기

문자열은 여러 개의 문자가 연결된 기차와 같습니다. 각 칸에는 번호가 있죠.

🚂 마스터 코치의 비유: 기차 좌석 번호
파이썬 기차는 0번부터 시작합니다! (1번 아님 주의!)
text[0]은 맨 앞 칸(첫 글자)을 의미합니다.
text[-1]은 맨 뒤 칸(마지막 글자)을 의미하는 역방향 티켓입니다!
0

▲ "Hello" 기차의 좌석 번호
text = "Hello"

print(text[0])   # 'H' (첫 번째 글자)
print(text[4])   # 'o' (마지막 글자)

# 뒤에서부터 셀 때는 마이너스(-)를 씁니다.
print(text[-1])  # 'o' (뒤에서 첫 번째)
print(text[-2])  # 'l' (뒤에서 두 번째)

2️⃣ 슬라이싱 (Slicing): 원하는 만큼 자르기

문자열을 통째로 쓰지 않고, 원하는 부분만 조각내서 가져올 수 있습니다.

🍰 마스터 코치의 비유: 케이크 자르기
text[시작:끝] 형태로 사용합니다.
중요한 규칙: 시작은 포함되지만, 끝 번호 바로 앞에서 멈춥니다!
마치 "3번 칸 앞에서 자르세요"라고 명령하는 것과 같아요. 
text = "Python Programming"

# 0번부터 6번 앞까지(5번까지) 가져오기
print(text[0:6])   # 'Python'

# 시작을 생략하면 처음부터!
print(text[:6])    # 'Python'

# 끝을 생략하면 끝까지!
print(text[7:])    # 'Programming'

3️⃣ 간격 (Step): 징검다리 건너기

하나씩 가져오지 않고, 퐁당퐁당 건너뛰며 가져올 수도 있습니다.

🐸 마스터 코치의 비유: 개구리 점프
text[시작:끝:간격]
간격(Step)만큼 점프하면서 글자를 가져옵니다.
-1을 쓰면? 뒤로 점프하니 거꾸로 뒤집기가 됩니다! 
numbers = "123456789"

# 2칸씩 점프! (홀수만 나오겠네요)
print(numbers[::2])   # '13579'

# 3칸씩 점프!
print(numbers[::3])   # '147'

# 뒤집기 마법! (Step이 -1이면 뒤에서부터)
text = "Hello"
print(text[::-1])  # 'olleH'

# 💡 고급 마법: 특정 구간만 거꾸로 가져오기!
# text[시작:끝:-1] -> 시작부터 '뒤로' 가서 끝 바로 앞(뒤쪽 기준)에서 멈춤
code = "Python"
# 인덱스: P(0) y(1) t(2) h(3) o(4) n(5)
print(code[5:1:-1]) 
# 결과: 'noht' (5번 'n'부터 시작해 1번 'y' 직전인 't'까지)

# 회문(Palindrome) 판별할 때 유용해요!
level = "level"
print(level == level[::-1]) # True (거꾸로 해도 같으니까요!)

4️⃣ 📝 인덱싱 & 슬라이싱 퀴즈

1. text = "Python" 일 때, print(text[1])의 결과는?
  • a. P
  • b. y
  • c. t
  • d. n
2. "Programming"에서 뒤에 3글자 "ing"만 가져오는 방법은?
  • a. s[-3:]
  • b. s[:-3]
  • c. s[3:]
  • d. s[::-3]
3. "ABCDE"에서 [1:4:2]의 결과는? (1번부터 4번 앞까지, 2칸씩)
  • a. ABC
  • b. BD
  • c. AC
  • d. CE
4. [코딩하기] 거꾸로 읽어도 같은 단어 만들기 
word = "LEVEL"
# 슬라이싱을 이용해 word를 거꾸로 뒤집어(reverse) 주세요.
reverse_word = word[_______]

print(reverse_word)

실행 결과: LEVEL

🔑 정답 확인하기 (클릭)
  1. b (y): 인덱스 0은 P, 1은 y입니다.
  2. a (s[-3:]): 뒤에서 3번째부터 끝까지 가져와야 합니다.
  3. b (BD): 1번(B) 가져오고, 2칸 점프해서 3번(D) 가져옵니다. 4번 앞까지이므로 E는 제외.
  4. ::-1: word[::-1]은 문자열을 처음부터 끝까지 역순으로 뒤집습니다.

📌 핵심 요약

  • 인덱싱(Indexing): [0]은 시작, [-1]은 끝. 기차 좌석 번호!
  • 슬라이싱(Slicing): [시작:끝]. 끝 번호는 포함되지 않아요! (케이크 자르기)
  • 간격(Step): [::2]는 징검다리 건너기, [::-1]은 뒤집기!

문자열을 자유자재로 다루는 당신은 마법사! 🧙‍♂️
