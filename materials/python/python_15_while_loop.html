1️⃣ WHILE 반복문: 될 때까지 해! 🥣

if문과 비슷하지만, 조건이 참(True)인 동안 계속해서 반복한다는 점이 다릅니다.

🥣 마스터 코치의 비유: 배부를 때까지 먹기
조건: "배가 고픈가요?" (True)
행동: 밥을 한 숟가락 먹는다.
(다시 확인) 배가 고픈가요? -> (True) -> 먹는다.
(다시 확인) 배가 고픈가요? -> (False) -> "잘 먹었습니다!" (종료) 
count = 0

# count가 3보다 작은 동안 계속 반복
while count < 3:
    print("안녕, Python!", count)
    count = count + 1  # 횟수를 1씩 늘려줌 (중요!)

print("끝!")

# 출력 결과:
# 안녕, Python! 0
# 안녕, Python! 1
# 안녕, Python! 2
# 끝!
🚨 경고: 무한 루프 (Infinite Loop)

만약 count = count + 1을 빼먹으면 어떻게 될까요?
count는 영원히 0이고, 조건은 영원히 True가 되어 프로그램이 멈추지 않습니다!
(마치 고장난 레코드판처럼요 💿)

※ 실수로 무한 루프에 빠졌다면 Ctrl + C를 눌러서 강제 종료하세요!

2️⃣ Random 모듈: 마법의 주사위 🎲

컴퓨터가 무작위로 숫자를 뽑게 하려면 random이라는 도구 상자를 가져와야 합니다.

🎲 마스터 코치의 비유: 주사위 굴리기
우리가 주사위를 던질 때 어떤 숫자가 나올지 모르는 것처럼,
컴퓨터에게 "아무 숫자나 하나 줘봐!"라고 부탁하는 것입니다.

사용법 (randint)

random.randint(a, b): a부터 b까지의 숫자 중 하나를 랜덤으로 뽑습니다. (b도 포함!)

import random  # 도구 상자 가져오기 (필수!)

# 1부터 6 사이의 랜덤 숫자 뽑기
dice = random.randint(1, 6)
print("주사위 눈:", dice)

# 실행할 때마다 결과가 달라집니다!

3️⃣ 실전 예제: 숫자 맞추기 게임 (Up & Down) 🎮

while문과 random을 섞으면 재밌는 게임을 만들 수 있습니다.
컴퓨터가 생각한 숫자를 맞출 때까지 반복합니다!

import random

secret = random.randint(1, 100) # 1~100 사이 랜덤 숫자
guess = 0                       # 사용자의 답 (초기화)

print("=== 숫자 맞추기 게임 시작! ===")

# 정답을 맞출 때까지(다를 동안) 계속 반복
while guess != secret:
    # 사용자에게 입력 받기 (문자를 숫자로 변환)
    guess = int(input("숫자를 입력하세요 (1-100): "))
    
    if guess < secret:
        print("Up! (더 큰 숫자입니다)")
    elif guess > secret:
        print("Down! (더 작은 숫자입니다)")
    else:
        print("정답입니다! 🎉")

print("게임 종료")

📌 핵심 요약

  • while 조건: 조건이 참인 동안 계속 반복합니다. (횟수를 모를 때 유용)
  • 무한 루프: 조건이 영원히 참이면 프로그램이 멈추지 않으니 조심해야 합니다.
  • random.randint(a, b): a부터 b까지의 랜덤한 정수를 뽑아줍니다.
  • import random: 랜덤 기능을 쓰려면 맨 위에 이 주문을 외워야 합니다!

이제 예측 불가능한 재미있는 프로그램도 만들 수 있습니다! 🎲
