🌌 파이썬 마스터 코치: 6. 연산자의 세계

컴퓨터는 계산기이자 논리 왕입니다! 그 비밀을 파헤쳐볼까요?

1️⃣ 참과 거짓 (Boolean)

파이썬에게는 딱 두 가지의 확고한 대답이 있습니다.

💡 마스터 코치의 비유: 전등 스위치
스위치는 켜짐(ON) 아니면 꺼짐(OFF) 뿐이죠?
파이썬도 마찬가지입니다.
True (참, 켜짐) 그리고 False (거짓, 꺼짐) 
print(True)   # 진짜!
print(False)  # 가짜!

# 비교를 하면 진실이 밝혀집니다.
print(10 > 5)  # 10은 5보다 크니? -> True
print(10 == 5) # 10이랑 5가 같니? -> False

2️⃣ 비교 연산자 (Comparison Operators)

누가 더 크고 작은지, 같은지 다른지 비교해 볼까요?

⚖️ 마스터 코치의 비유: 양팔 저울
양쪽의 무게(값)를 비교해서 결과를 알려줍니다.
결과는 무조건 True(참) 아니면 False(거짓)입니다!
연산자 의미 예시 결과
== 같다 10 == 10 True
!= 다르다 10 != 5 True
> 크다 (초과) 10 > 5 True
< 작다 (미만) 10 < 5 False
>= 크거나 같다 (이상) 10 >= 10 True
<= 작거나 같다 (이하) 10 <= 5 False

3️⃣ 산술 연산자 (Arithmetic Operators)

더하기, 빼기 말고도 재미있는 연산자가 많아요.

🧮 마스터 코치의 비유: 슈퍼 계산기
// (버림 나눗셈): 소수점 뒤 꼬리를 싹뚝 자르는 가위!
% (나머지): 몫은 버리고 찌꺼기만 챙기는 알뜰함!
** (거듭제곱): 숫자가 순식간에 뻥튀기되는 마법!
연산자 설명 예시 (a=10, b=3)
+ 더하기 10 + 3 = 13
- 빼기 10 - 3 = 7
* 곱하기 10 * 3 = 30
/ 나누기 (실수) 10 / 3 = 3.333...
// 나누기 (정수, 몫) 10 // 3 = 3 (소수점 버림)
% 나머지 10 % 3 = 1
** 거듭제곱 10 ** 3 = 1000

✨ 마법의 지름길 (할당 연산자)

귀찮은 건 딱 질색이죠? 코드를 줄여쓰는 방법이 있습니다.

  • a = a + 10 👉 a += 10 (더하고 저장!)
  • a = a - 5 👉 a -= 5 (빼고 저장!)
  • a = a * 2 👉 a *= 2 (곱하고 저장!)

4️⃣ 논리 연산자 (Logical Operators)

여러 가지 조건을 한 번에 따져봐야 할 때 사용합니다.

🛡️ AND (엄격한 문지기): "초대장 그리고(and) 신분증 둘 다 가져와! 아니면 못 들어가!" 
print(True and True)   # 둘 다 참이어야 True
print(True and False)  # 하나라도 거짓이면 False
👐 OR (관대한 문지기): "현금 또는(or) 카드 중 하나만 내! 다 받아줄게!" 
print(True or False)   # 하나만 참이어도 True
print(False or False)  # 둘 다 거짓이어야 False
🔄 NOT (청개구리 거울): "참이면 거짓으로, 거짓이면 참으로 뒤집어!" 
print(not True)    # False
print(not False)   # True

5️⃣ 📝 연산자 능력 시험

1. 다음 중 '나눗셈의 몫'만 구하는 연산자는?
  • a. /
  • b. %
  • c. //
  • d. **
2. 다음 코드의 실행 결과는? 
print(10 > 20 or 10 == 10)
  • a. True
  • b. False
  • c. Error
  • d. None
3. 다음 중 결과가 True인 것은?
  • a. not True
  • b. True and False
  • c. 10 != 10
  • d. 5 > 3
4. [코딩하기] 놀이공원에 오신 것을 환영합니다!

키가 120cm 이상이어야 롤러코스터를 탈 수 있습니다.
아래 코드를 완성하여 탑승 가능 여부(True/False)를 출력하세요.

height = 115
limit = 120

# 키(height)가 제한(limit)보다 크거나 같은지 확인
print(height ____ limit)

실행 결과: False

🔑 정답 확인하기 (클릭)
  1. c: //는 몫을, %는 나머지를 구합니다.
  2. a: False or True 이므로 결과는 True(참)입니다.
  3. d: 5는 3보다 큽니다. (a:False, b:False, c:False)
  4. >=: 크거나 같다는 >= 연산자를 사용합니다. (115 >= 120 은 False)

📌 핵심 요약

  • Boolean: True(참), False(거짓).
  • 산술: //(몫), %(나머지), **(거듭제곱) 기억하기.
  • 논리: and(모두 참), or(하나만 참), not(뒤집기).
