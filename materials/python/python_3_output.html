💜 파이썬 마스터 코치: 3. 출력 방법

print 함수의 숨겨진 능력들을 알아볼까요?

1️⃣ 값 여러 개 출력하기

print() 함수에 콤마(,)를 사용하면 여러 값을 한 번에 출력할 수 있습니다.

🚃 마스터 코치의 비유: 기차 연결하기
여러 데이터를 기차 객차처럼 쭉 연결해서 내보낼 수 있어요.
기본적으로는 객차 사이에 공백(Space)이라는 연결 고리가 자동으로 들어갑니다!

💻 코드 예제

print(1, 2, 3)
# 출력: 1 2 3 (사이사이에 공백이 들어감)

print('안녕', '파이썬')
# 출력: 안녕 파이썬

a, b = 10, 20
print("결과:", a, b)
# 출력: 결과: 10 20

💡 잠깐! 매개변수(Parameter)가 뭔가요?

이제부터 배울 sep과 end는 print() 함수의 매개변수(Parameter)라고 부릅니다.

⚙️
옵션 설정하기
함수에게 "원래 하던 대로 하지 말고, 내가 원하는 대로 해줘!" 라고
구체적인 옵션을 설정해주는 것이라고 생각하면 쉬워요.

(예: "짜장면 주세요" (기본) vs "짜장면 주세요, 오이 빼고(옵션)")

2️⃣ sep 옵션 (구분자 바꾸기)

값들 사이에 들어가는 공백(기본 띄어쓰기)이 싫다면? sep으로 바꿀 수 있어요!

🔗 마스터 코치의 비유: 연결 고리(접착제) 바꾸기
기본적으로는 '투명 테이프(공백)'로 붙이지만,
우리가 원하면 '강력 본드(없음)'나 '예쁜 스티커(특수문자)'로 붙일 수 있어요.

💻 코드 예제

# 콤마(,)로 연결하고 싶을 때
print(1, 2, 3, sep=', ')
# 출력: 1, 2, 3

# 딱 붙이고 싶을 때 (빈 문자열)
print('안녕', '파이썬', sep='')
# 출력: 안녕파이썬

# 특수문자로 연결
print(2025, 3, 1, sep='-')
# 출력: 2025-3-1

3️⃣ end 옵션 (끝처리 바꾸기)

print()는 출력이 끝나면 자동으로 줄바꿈(Enter)을 합니다. 이걸 end로 바꿀 수 있어요.

🏁 마스터 코치의 비유: 마무리 포즈
무대(출력)가 끝나고 기본적으로는 퇴장(다음 줄로 이동)하지만,
퇴장하지 않고 무대 위에 그대로 서 있거나(줄바꿈 안 함), 다른 포즈(문자 출력)를 취할 수도 있어요.

💻 코드 예제

# 기본: 줄바꿈이 일어남
print("첫 번째 줄")
print("두 번째 줄")
# 출력:
# 첫 번째 줄
# 두 번째 줄

# 줄바꿈 없이 이어서 출력하기
print("안녕", end='')
print("하세요")
# 출력: 안녕하세요 (두 줄이었지만 한 줄로 나옴!)

# 끝에 마침표 찍고 줄바꿈 대신 띄어쓰기
print("사과", end=' ')
print("포도", end=' ')
print("배")
# 출력: 사과 포도 배

4️⃣ 📝 출력 마스터 퀴즈

1. 다음 코드의 실행 결과는? 
print(10, 20, 30, sep="&")
  • a. 10 20 30
  • b. 10&20&30
  • c. 10, 20, 30
  • d. 102030
2. 두 문장을 한 줄에 이어서 출력하려면 빈칸에 무엇을 넣어야 할까요? 
print("Hello", ____="")
print("World")
  • a. sep
  • b. start
  • c. end
  • d. next
3. 다음 중 010-1234-5678 을 출력하는 올바른 코드는?
  • a. print("010", "1234", "5678", sep="-")
  • b. print("010", "1234", "5678", end="-")
  • c. print("010", "1234", "5678")
  • d. print("010" + "1234" + "5678", sep="-")
🔑 정답 확인하기 (클릭)
  1. b. 10&20&30: sep은 값들 사이를 채워줍니다.
  2. c. end: print가 끝난 후 줄바꿈을 하지 않으려면 end 옵션을 사용합니다.
  3. a: 값들 사이에 하이픈(-)을 넣으려면 sep 옵션이 필요합니다. (d는 이미 다 붙여버려서 sep이 소용없음)

📌 핵심 요약

  • 여러 개 출력: print(값1, 값2) → 기본적으로 공백으로 띄워짐.
  • sep="...": 값들 사이(between)에 넣을 문자 지정. (예: sep=",")
  • end="...": 출력이 끝난 후(after)에 할 행동 지정. (예: end="" 줄바꿈 금지)

이제 원하는 대로 출력을 조종할 수 있어요! 🎮
