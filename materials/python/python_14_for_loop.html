🔵 파이썬 마스터 코치: 14. 반복문 (for)

엄청난 양의 작업도 한 방에! 컴퓨터의 성실함을 배워봅시다.

1️⃣ FOR 반복문: 차례대로 하나씩! 🏭

리스트나 문자열 등에 있는 요소들을 처음부터 끝까지 하나씩 꺼내서 처리할 때 사용합니다.

🏭 마스터 코치의 비유: 컨베이어 벨트
리스트(벨트) 위에 물건들이 놓여 있습니다.
작업자(for 변수)가 물건을 하나씩 집어들고(`in`) 작업을 수행합니다.
물건이 다 떨어지면 퇴근합니다! (반복 종료) 
fruits = ["사과", "바나나", "포도"]

# "fruits" 상자에서 과일을 하나씩 꺼내 "fruit"라고 부르며 작업
for fruit in fruits:
    print("나는 " + fruit + "를 좋아해!")

# 출력 결과:
# 나는 사과를 좋아해!
# 나는 바나나를 좋아해!
# 나는 포도를 좋아해!

2️⃣ range() 함수: 번호표 뽑기 🎫

숫자를 자동으로 만들어주는 마법의 기계입니다. for문과 찰떡궁합입니다.

🎫 마스터 코치의 비유: 번호표 기계
range(5): 0번부터 5개 뽑아줘! (0, 1, 2, 3, 4)
주의: 마지막 숫자는 포함하지 않습니다! (5 직전까지만)

사용법 3가지

  • range(stop): 0부터 stop-1까지 (1씩 증가)
  • range(start, stop): start부터 stop-1까지 (1씩 증가)
  • range(start, stop, step): start부터 stop-1까지 (step만큼 점프!)
# 1. 5번 반복 (0부터 4까지)
for i in range(5):
    print(i, "번째 인사")

# 2. 2부터 5 전까지 (2, 3, 4)
for num in range(2, 5):
    print(num)

# 3. 1부터 10 전까지 2칸씩 점프 (1, 3, 5, 7, 9)
for odd in range(1, 10, 2):
    print(odd)
⚠️ 주의: 숫자가 줄어들 때는?

range(10, 0)처럼 쓰면 아무 일도 일어나지 않습니다. 기본적으로 숫자는 1씩 증가하기 때문입니다.
거꾸로 세려면 -1(감소)이라고 콕 집어 알려줘야 합니다!

# 🛑 작동 안 함 (아무것도 출력 안 됨)
for i in range(10, 0):
    print("안녕, Python!", i)

# ✅ 작동 함 (5부터 1까지)
for i in range(5, 0, -1):
    print("안녕, Python!", i)
    
# 출력 결과:
# 안녕, Python! 5
# 안녕, Python! 4... (1까지)
💡 꿀팁: 거꾸로 뒤집기 (reversed)

일일이 -1을 계산하기 귀찮다면? reversed() 함수로 감싸주면 알아서 뒤집어줍니다!

# 1. 0부터 4까지 뒤집기 (4, 3, 2, 1, 0)
for i in reversed(range(5)):
    print(i)

# 2. 1부터 6 전까지 뒤집기 (5, 4, 3, 2, 1)
for i in reversed(range(1, 6)):
    print(i)

# 3. 1부터 10 전까지 2칸씩 뛴 걸 뒤집기 (9, 7, 5, 3, 1)
for i in reversed(range(1, 10, 2)):
    print(i)

3️⃣ 이중 반복문 (Double Loop): 뫼비우스의 띠? 🕰️

반복문 안에 또 반복문이 있는 형태입니다. 구구단이나 시계를 생각하면 쉽습니다.

🕰️ 마스터 코치의 비유: 시계 바늘
큰 바늘(분침)이 60번 돌아야,
작은 바늘(시침)이 겨우 1칸 움직입니다.
안쪽 루프가 다~ 끝나야 바깥쪽 루프가 한 번 다음으로 넘어갑니다. 
# 구구단 2단부터 4단까지 출력
for dan in range(2, 5):        # 바깥쪽 루프 (단)
    print("=== " + str(dan) + "단 ===")
    for hang in range(1, 10):  # 안쪽 루프 (곱해지는 수 1~9)
        print(dan, "x", hang, "=", dan * hang)
    # 안쪽 루프가 9번 다 돌아야 여기로 나옴

4️⃣ 📝 반복문 마스터 테스트

1. range(3)이 만들어내는 숫자는?
  • a. 1, 2, 3
  • b. 0, 1, 2, 3
  • c. 0, 1, 2
  • d. 1, 2
2. 다음 코드의 출력 횟수는? 
for i in range(2):      # 2번 반복
    for j in range(3):  # 3번 반복
        print("안녕")
  • a. 2번
  • b. 3번
  • c. 5번 (2+3)
  • d. 6번 (2x3)
3. [코딩하기] 카운트다운!

10부터 1까지 거꾸로 센 뒤 "발사!"를 외쳐보세요.

# 힌트: range(start, stop, step) 활용
for i in range(__, __, __):
    print(i)
print("발사!")
4. [코딩하기] 1부터 100까지 더하기

1부터 100까지의 모든 숫자를 더한 값을 구해봅시다.

total = 0
for i in range(1, 101):
    total = total + _____
print(total) # 결과: 5050
🔑 정답 확인하기 (클릭)
  1. c: 0부터 시작하고, 3은 포함하지 않습니다. (0, 1, 2)
  2. d: 바깥쪽 1번당 안쪽 3번 실행되므로 총 2 x 3 = 6번입니다.
  3. 코딩 3: range(10, 0, -1) (10부터 0 전까지 -1씩 감소)
  4. 코딩 4: total + i (현재 숫자 i를 total에 계속 누적)

📌 핵심 요약

  • for 변수 in 리스트: 리스트에 있는 요소를 하나씩 꺼내쓰기 좋습니다.
  • range(n): 0부터 n-1까지 숫자를 만들어냅니다. (횟수 반복에 최고!)
  • 이중 반복문: 반복 안에 반복이 들어갑니다. (구구단, 시계바늘 원리)
  • 들여쓰기: 반복할 코드도 if문처럼 꼭 들여쓰기를 해야 합니다.
