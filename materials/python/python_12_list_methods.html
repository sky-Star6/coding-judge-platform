🟢 파이썬 마스터 코치: 12. 리스트 조작

넣고, 빼고, 정리하는 스마트한 정리 정돈 기술을 익혀보세요!

1️⃣ 추가하기: 장바구니 담기

리스트에 새로운 아이템을 추가하는 방법들입니다.

🛒 마스터 코치의 비유: 쇼핑
append(): 계산대 위에 물건을 하나씩 맨 뒤에 올려놓습니다.
insert(): 새치기처럼 원하는 위치(인덱스)에 끼워 넣습니다.
extend(): 다른 장바구니에 있는 물건들을 몽땅 쏟아붓습니다. 
cart = ["사과", "바나나"]

# 1. 맨 뒤에 추가 (append)
cart.append("체리")
print(cart) # ['사과', '바나나', '체리']

# 2. 중간에 끼워넣기 (insert)
# 1번 인덱스(두 번째) 자리에 '딸기' 추가
cart.insert(1, "딸기")
print(cart) # ['사과', '딸기', '바나나', '체리']

# 3. 리스트 합치기 (extend)
cart.extend(["포도", "수박"])
print(cart) # ['사과', '딸기', '바나나', '체리', '포도', '수박']

2️⃣ 삭제하기: 물건 꺼내기/버리기

리스트에서 아이템을 제거하는 방법들입니다.

🗑️ 마스터 코치의 비유: 정리 정돈
pop(): 물건을 꺼내서 사용하고 없앱니다. (맨 뒤 또는 콕 집어서)
remove(): 특정 물건을 찾아서 버립니다. (값이 기준)
clear(): 장바구니를 완전히 비웁니다.
del: 강력한 삭제! 위치(인덱스)나 범위(슬라이싱)로 지워버립니다. 
todos = ["청소", "빨래", "공부", "운동"]

# 1. 꺼내기 (pop) - 맨 뒤(운동)를 쏙 뺌
done = todos.pop()
print(done)    # "운동"
print(todos)   # ['청소', '빨래', '공부']

# 1-1. 특정 위치 꺼내기 (pop(index))
first = todos.pop(0)
print(first)   # "청소" (맨 앞 꺼냄)

# 2. 찾아서 지우기 (remove) - "빨래"를 찾아 삭제
todos.remove("빨래")
print(todos)   # ['공부']

# 3. 싹 비우기 (clear)
todos.clear()
print(todos)   # []

# 4. 강력한 삭제 (del)
fruits = ["사과", "망고", "체리", "포도", "수박"]

# 4-1. 인덱스로 삭제
del fruits[1]   # 1번(망고) 삭제
print(fruits)   # ['사과', '체리', '포도', '수박']

# 4-2. 슬라이싱으로 삭제 (범위 삭제)
del fruits[1:3] # 1번부터 3번 전까지 (체리, 포도) 삭제
print(fruits)   # ['사과', '수박']

3️⃣ 정렬하기: 책꽂이 정리

순서를 뒤바꾸거나 규칙대로 나열합니다.

📚 마스터 코치의 비유: 도서관 사서
sort(): 가나다(ABC) 순서대로 원본을 바꿈 (파괴적).
reverse(): 현재 순서를 거꾸로 뒤집음 (정렬 아님, 그냥 뒤집기).
sorted(): 원본은 그대로 두고, 정렬된 복사본을 줌 (안전). 
nums = [3, 1, 5, 2, 4]

# 1. 정렬하기 (sort) - 원본이 바뀜!
nums.sort()
print(nums)   # [1, 2, 3, 4, 5] (오름차순)

# 1-1. 내림차순 정렬 (거꾸로)
nums.sort(reverse=True)
print(nums)   # [5, 4, 3, 2, 1] (큰 수부터)

# 2. 뒤집기 (reverse)
nums.reverse()
print(nums)   # [5, 4, 3, 2, 1]

# 3. 뒤집기 (슬라이싱) - 원본 유지
# [::-1]은 뒤에서부터 끝까지 가져옵니다.
original = [1, 2, 3]
reversed_copy = original[::-1]
print(reversed_copy) # [3, 2, 1]

# 번외: sorted() - 원본 유지
data = [9, 7, 8]
new_data = sorted(data)
print(data)      # [9, 7, 8] (그대로)
print(new_data)  # [7, 8, 9] (정렬됨)

4️⃣ 위치 찾기와 개수 세기

문자열 메서드와 똑같은 이름을 가진 친구들입니다.

animals = ["dog", "cat", "dog", "bird"]

# 위치 찾기 (index)
print(animals.index("cat"))  # 1

# 개수 세기 (count)
print(animals.count("dog"))  # 2

# 💡 뒤에서부터 찾기 (rfind 없음!)
# 리스트는 rfind()나 rindex() 함수가 없습니다.
# 대신 뒤집어서 찾은 뒤 인덱스를 계산해야 합니다.
my_list = [1, 2, 3, 2, 1]
target = 2

# 전체 길이 - 1 - (뒤집은 리스트에서의 인덱스)
last_index = len(my_list) - 1 - my_list[::-1].index(target)
print(last_index)  # 결과: 3

5️⃣ 튜플의 조작: "만지지 마세요!"

튜플은 생성된 후에는 변경할 수 없습니다(Immutable).

🚫 튜플은 이런 함수가 없습니다:
append, insert, pop, remove, sort... (전부 에러 발생!)

✅ 가능한 것:
index(), count() (내용을 바꾸지 않는 조회 기능만 가능)

6️⃣ 존재 여부 확인: 있니? 없니?

리스트나 튜플 안에 특정 값이 있는지 확인할 때 in 연산자를 사용합니다.

cart = ["사과", "바나나"]

print("사과" in cart)  # True (있어요)
print("포도" in cart)  # False (없어요)
print("포도" not in cart) # True (없는 게 맞아요)

7️⃣ 꿀팁: 리스트의 할당과 복사 🍯

할당과 복사는 비슷한 것 같지만 큰 차이점이 있습니다.

👯 마스터 코치의 비유: 쌍둥이 vs 그림자
할당(Assignment): 나(원본)와 그림자(할당된 변수)입니다. 내가 움직이면 그림자도 같이 움직입니다.
복사(Copy): 나와 똑같이 생긴 복제인간(클론)을 만듭니다. 내가 다쳐도 복제인간은 멀쩡합니다.

1. 할당 (위험!)

b = a와 같이 리스트를 다른 변수에 할당하면 리스트는 두 개가 될 것 같지만 실제로는 한 개입니다.
b[2] = 99와 같이 리스트 b 의 값을 변경하면 리스트 a와 b에 모두 반영이 됩니다. 

a = [0, 0, 0, 0, 0]
b = a # 할당 (이름표만 하나 더 붙임)

b[2] = 99
print(a) # [0, 0, 99, 0, 0] (a도 바뀜!)
print(b) # [0, 0, 99, 0, 0]

# 연산자로 비교를 해보면 두 객체는 같다.
a = [0, 0, 0, 0, 0]
b = a
print(a is b) # True (완전 같은 녀석)
print(a == b) # True (값도 같음)

2. 복사 (안전!)

여기서 리스트를 완전히 두 개로 만들려면 copy()로 요소를 복사해야 합니다.

a = [0, 0, 0, 0, 0]
b = a.copy() # 복사 (쌍둥이 생성)

b[2] = 99
print(a) # [0, 0, 0, 0, 0] (a는 무사함!)
print(b) # [0, 0, 99, 0, 0] (b만 바뀜)

# 연산자로 비교를 해보면 두 객체는 다르다.
a = [0, 0, 0, 0, 0]
b = a.copy()
print(a is b) # False (다른 녀석임)
print(a == b) # True (내용물은 같음)

  • a. [1, 2, 3, 4]
  • b. [1, 2, [3, 4]]
  • c. [1, 2, 3]
  • d. 에러 발생
2. 리스트에서 값 '10'을 삭제하고 싶을 때 쓰는 함수는?
  • a. del(10)
  • b. pop(10)
  • c. remove(10)
  • d. delete(10)
3. [코딩하기] 대기열 관리

손님이 왔습니다(append). 맨 앞 손님이 입장합니다(pop(0)).

queue = ["철수", "영희"]
# "민수" 도착
queue.________("민수")

# "철수" 입장 (맨 앞 사람)
first = queue.pop(____)

print(queue) 
# 결과: ['영희', '민수']
🔑 정답 확인하기 (클릭)
  1. b: append는 덩어리째 넣습니다. 낱개로 합치려면 extend!
  2. c: remove(값), pop(인덱스)
  3. append, 0: first = queue.pop(0)

📌 핵심 요약

  • 추가: append(맨 뒤), insert(중간), extend(확장).
  • 삭제: pop(꺼내기), remove(값 삭제), clear(비우기).
  • 정렬: sort(원본 정렬), reverse(뒤집기), sorted(새 정렬).
  • 튜플: 조회(index, count)만 가능, 수정 불가!
