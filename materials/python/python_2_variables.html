🎁 파이썬 마스터 코치: 변수와 자료형

변수는 데이터를 담는 마법 상자! 어떤 데이터인지도 알아봐요.

1️⃣ 변수 (Variable) 만들기

프로그래밍에서 가장 기본이 되는 데이터를 저장하는 공간입니다.

📦 마스터 코치의 비유: 마법 상자
변수는 이름이 붙은 상자입니다.
score = 100은 "score라는 이름의 상자를 만들고, 그 안에 숫자 100을 넣어라!" 라는 뜻이에요.
할당

📝 기본 문법: 할당(Assignment)

수학시간에 배운 = 기호와 다릅니다!
"오른쪽에 있는 값을 왼쪽 상자에 넣는다"는 뜻입니다.

x = 10       # x 상자에 10을 넣음
print(x)     # 출력: 10

x = 30 + 40  # 30+40의 결과인 70을 x 상자에 넣음 (기존 10은 사라짐!)
print(x)     # 출력: 70

✨ 변수의 특별한 기술들

1. 여러 개 한 번에 만들기

x, y, z = 10, 20, 30
print(x, y, z) # 출력: 10 20 30

2. 두 변수의 값 맞바꾸기 (Swapping)

다른 언어에서는 임시 상자가 필요하지만, 파이썬은 한 방에 가능합니다!

a = 10
b = 20
a, b = b, a  # 서로 교환!
print(a, b)  # 출력: 20 10

3. 변수 삭제하기

x = 10
del x        # x 상자를 아예 없애버림
# print(x) -> 에러 발생! (그런 상자 없는데요?)

2️⃣ 자료형 (Data Type)

변수 상자에 담을 수 있는 데이터의 종류입니다.

🍰 마스터 코치의 비유: 음식의 종류
상자에 케이크(숫자)를 넣을 수도 있고, 쪽지(문자)를 넣을 수도 있죠.
파이썬은 눈치가 빨라서 우리가 넣는 내용을 보고 "아, 이건 케이크구나!" 하고 알아서 타입을 정해줍니다. (동적 타이핑)

📊 주요 자료형 총정리

분류 자료형 (Type) 비유 (Metaphor) 설명
숫자형 int (정수) 🎂 홀 케이크 소수점 없는 숫자 (예: 2025)
float (실수) 🍰 조각 케이크 소수점 있는 숫자 (예: 3.14)
문자열 str 📜 텍스트 쪽지 글자들의 나열 (예: "Hello")
불린형 bool 💡 스위치 참(True) / 거짓(False)
시퀀스 list (리스트) 🎒 책가방 순서대로 마구 담을 수 있음 (수정 가능)
tuple (튜플) 🔒 투명 금고 순서대로 담지만, 내용 변경 불가
매핑형 dict (딕셔너리) 📒 전화번호부 이름(Key)-값(Value) 쌍으로 저장
세트형 set (세트) 🧺 빨래 바구니 순서 없고 중복 허용 안 함
없음 None 🕳️ 빈 공간 "값이 없음"을 나타냄

💻 코드 예제

year = 2025                # int
gpa = 4.5                  # float
name = "파이썬"             # str
is_happy = True            # bool
scores = [90, 80, 70]      # list
location = (37.5, 127.0)   # tuple
me = {"name": "Mario"}     # dict
colors = {"red", "red"}    # set -> {"red"} (중복제거)
empty = None               # NoneType

3️⃣ 📝 변수 만들기 문제

1. 다음 중 변수를 만드는 방법으로 올바른 것을 고르세요.
  • a. int x = 10
  • b. x, 10
  • c. x = 10
  • d. x <- 10
  • e. x := 10
2. 다음 중 변수 이름으로 사용할 수 없는 것을 모두 고르세요.
  • a. 300 (숫자로 시작)
  • b. _53A
  • c. if (키워드)
  • d. z
  • e. H-80 (하이픈 사용 불가)
  • f. _hello3
  • g. 30seconds (숫자로 시작)
  • h. 99% (특수문자 %)
3. 파이썬에서 "아무것도 없음"을 나타내는 자료형은?
  • a. NULL
  • b. null
  • c. nil
  • d. None
  • e. 0
4. 변수 여러 개를 만드는 방법으로 올바른 것은? (모두 고르시오)
  • a. a, b, c = None
  • b. a = b = c = 10
  • c. a, b, c = 10, 20 (개수 안 맞음)
  • d. a, b, c = 10, None, 30
  • e. a, b = 10, 20, None (개수 안 맞음)
5. 빈칸을 채워 아래 결과가 나오게 하세요. 
a = b = ①
c, d = ②
print(a) # 10
print(b) # 10
print(c) # 20
print(d) # None
🔑 정답 확인하기 (클릭)
  1. c. x = 10: 파이썬은 타입 지정 없이 바로 할당합니다.
  2. a, c, e, g, h:
    • a, g: 숫자로 시작 불가
    • c: 키워드(if)
    • e, h: 허용되지 않는 특수문자(-, %)
  3. d. None: 파이썬의 Null 값은 None입니다.
  4. b, d:
    • b: 체인 할당 (모두 10)
    • d: 1:1 매칭 할당 (a=10, b=None, c=30)
  5. ① 10, ② 20, None: 
    a = b = 10
c, d = 20, None

📌 핵심 요약

  • 변수 선언: 변수명 = 값 (오른쪽 값을 왼쪽 상자에!)
  • 다중 할당: a, b = 1, 2 또는 a = b = 10
  • 값 교환: a, b = b, a (임시 변수 필요 없음)
  • 자료형: 파이썬은 값이 들어갈 때 자동으로 타입을 정해줍니다. (int, float, str, list ...)
