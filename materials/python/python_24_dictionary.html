🔵 파이썬 마스터 코치: 24. 딕셔너리

순서보다 중요한 건 이름표! 🏷️ 데이터를 똑똑하게 보관하세요.

1️⃣ 딕셔너리(Dictionary)란? 스마트 사물함 🔐

리스트처럼 여러 값을 담지만, 번호(인덱스) 대신 이름표(Key)를 붙여서 관리합니다. (순서는 중요하지 않아요!)

🔐 마스터 코치의 비유: 사물함
사물함에 물건(Value)을 넣고, 각 사물함마다 이름표(Key)를 붙입니다.
이름표만 알면, 몇 번째 칸인지 몰라도 바로 찾을 수 있습니다!
"name" ➡ "Iron Man"
"age" ➡ 45 
# 중괄호 {} 사용, Key: Value 형태
hero = {
    "name": "Iron Man",
    "age": 45,
    "weapon": "Suit"
}

2️⃣ 접근하기: 열쇠로 열기 🔑

리스트는 a[0]처럼 숫자를 쓰지만, 딕셔너리는 d["Key"]처럼 키를 사용합니다.

# 1. 대괄호 접근 (열쇠로 바로 열기)
print(hero["name"])  # Iron Man
# print(hero["address"]) # 🚨 Error! 없는 키를 부르면 오류 발생

# 2. get() 메서드 사용 (똑똑하게 열기)
print(hero.get("name")) # Iron Man
print(hero.get("address")) # None (오류 없이 '없음'을 알려줌)

3️⃣ 추가 및 수정하기: 사물함 관리 🛠️

새로운 키를 쓰면 추가되고, 있는 키를 쓰면 수정됩니다. (사용법이 똑같아요!)

# 값 수정하기 (이미 있는 키)
hero["age"] = 50
print(hero["age"]) # 50

# 값 추가하기 (새로운 키)
hero["color"] = "Red"
print(hero)
# {'name': 'Iron Man', 'age': 50, 'weapon': 'Suit', 'color': 'Red'}

4️⃣ 삭제하기: 사물함 비우기 🗑️

필요 없는 데이터는 지워버릴 수 있습니다.

# 1. del 키워드 (완전 삭제)
del hero["color"]

# 2. pop() 메서드 (꺼내서 확인하고 삭제)
weapon = hero.pop("weapon")
print(weapon) # Suit (꺼낸 값)
print(hero)   # {'name': 'Iron Man', 'age': 50} (weapon 사라짐)

4️⃣-2. 딕셔너리 더 잘 쓰기 (심화) 🚀

프로그래머들이 매일 쓰는 아주 유용한 기능 3가지입니다.

1. 내용물 한 번에 꺼내기 (반복문 전용)
  • keys(): 이름표(Key)만 다 가져와!
  • values(): 내용물(Value)만 다 가져와!
  • items(): 이름표랑 내용물을 쌍(Tuple)으로 다 가져와! (👍 추천)
hero = {"name": "Iron Man", "age": 50}

# items()를 쓰면 키와 값을 변수 두 개로 한 번에 받습니다.
for key, value in hero.items():
    print(key, ":", value)
# name : Iron Man
# age : 50
2. 있나 없나 확인하기 (`in` 연산자)
없는 키를 꺼내면 에러가 나니까, 미리 확인하는 습관이 중요합니다. 
if "address" in hero:
    print(hero["address"])
else:
    print("주소 정보가 없습니다.")
3. 싹 비우기 (`clear()`)
딕셔너리 통 자체는 남겨두고, 내용물만 깨끗이 설거지합니다. 
hero.clear()
print(hero) # {} (빈 딕셔너리)

📌 핵심 요약

  • 딕셔너리: {Key: Value} 쌍으로 이루어진 데이터.
  • 접근: d["키"] 또는 d.get("키") (안전).
  • 추가/수정: d["키"] = 값 (키가 없으면 추가, 있으면 수정).
  • 삭제: del d["키"] 또는 d.pop("키").
