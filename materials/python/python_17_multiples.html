🟠 파이썬 마스터 코치: 17. 배수와 공배수

규칙적인 숫자의 리듬을 찾아봅시다! 짝짝짝 👏

1️⃣ 배수 구하기: 징검다리 건너기 🐾

어떤 수를 나누었을 때 나머지가 0이 되는 수를 찾으면 됩니다.

🐾 마스터 코치의 비유: 징검다리
3의 배수는 3칸씩 점프해서 밟는 돌과 같습니다.
(3, 6, 9, 12...)
핵심 도구: 나머지 연산자 %
숫자 % 3 == 0 (3으로 나눈 나머지가 0이면 3의 배수!) 
# 1부터 20까지 숫자 중 3의 배수만 찾기
for i in range(1, 21):
    if i % 3 == 0:
        print(i, "짝 👏")
    
# 출력 결과: 3, 6, 9, 12, 15, 18

2️⃣ 공배수 구하기: 만남의 광장 🤝

두 가지 조건을 동시에 만족(and)하는 숫자를 찾습니다.

🤝 마스터 코치의 비유: 약속 장소
철수는 3일마다, 영희는 5일마다 공원에 갑니다.
둘이 만나는 날은 언제일까요?
3의 배수 이면서(and) 5의 배수인 날입니다! 
# 1부터 50까지 중 3과 5의 공배수 (15의 배수) 찾기
for i in range(1, 51):
    # 3의 배수 AND 5의 배수
    if i % 3 == 0 and i % 5 == 0:
        print(i, "찾았다! 공배수")

# 출력 결과:
# 15 찾았다! 공배수
# 30 찾았다! 공배수
# 45 찾았다! 공배수

📌 핵심 요약

  • 배수 확인: 나머지(%) 연산자가 0인지 확인합니다.
  • 공배수 확인: and 연산자로 두 조건을 묶어줍니다.
  • 이 원리를 알면 짝수/홀수 구분부터 복잡한 주기 계산까지 가능합니다!
