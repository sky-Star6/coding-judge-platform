🟫 파이썬 마스터 코치: 21. 2차원 리스트

리스트 안에 리스트가? 데이터로 아파트 건물을 지어봅시다! 🏢

1️⃣ 2차원 리스트란? 아파트 짓기 🏢

리스트([]) 안에 또 다른 리스트가 들어있는 형태입니다. 표(행, 열)처럼 데이터를 관리할 때 사용합니다.

🏢 마스터 코치의 비유: 아파트 건물
일반 리스트가 긴 기차(1차원)라면,
2차원 리스트는 층(행)과 호수(열)가 있는 아파트입니다. 
# 3층짜리 아파트 (각 층에 3가구씩)
apt = [
    [101, 102, 103],  # 0층 (1행)
    [201, 202, 203],  # 1층 (2행)
    [301, 302, 303]   # 2층 (3행)
]

2️⃣ 데이터 접근하기: 몇 층, 몇 호? 🔑

값을 가져오려면 두 번 찾아가야 합니다. 리스트[행][열]

행 \ 열 0열 (0호) 1열 (1호) 2열 (2호)
0행 (0층) 101 102 103
1행 (1층) 201 202 🔴 203 
# 질문: 202호를 찾아가려면?
# 1. 1번 층(가운데 줄)으로 간다 -> apt[1]
# 2. 거기서 1번 방(가운데 칸)으로 간다 -> apt[1][1]

print(apt[0])     # [101, 102, 103] -> 0층 전체
print(apt[1][1])  # 202 -> 1층의 1호

3️⃣ 전체 출력하기: 경비원 순찰 👮‍♂️

모든 방을 확인하려면 이중 반복문(for문 2개)이 필요합니다.

👮‍♂️ 마스터 코치의 비유: 순찰
경비원 아저씨가 한 층(바깥 for문)에 내려서,
그 층의 모든 복도(안쪽 for문)를 훑습니다. 
for floor in apt:          # 층별로 반복 (큰 덩어리)
    for room in floor:     # 호수별로 반복 (작은 데이터)
        print(room, end=" ")
    print() # 한 층 순찰 끝나면 줄바꿈

# 출력 결과:
# 101 102 103
# 201 202 203
# 301 302 303

3️⃣-2. 다양한 반복문 활용 🔄

상황에 따라 for문 뿐만 아니라 while문이나 인덱스를 직접 사용할 수도 있습니다.

# 예제 데이터: 3행 2열 리스트
numbers = [[10, 20], [30, 40], [50, 60]]

# 1. for문과 Unpacking (변수 두 개로 한 번에 받기)
for x, y in numbers: # 리스트의 가로 한 줄(안쪽 리스트)에서 값 두 개를 꺼냄
    print(x, y)

print("-" * 20)

# 2. for문과 range (인덱스로 접근하기)
for i in range(len(numbers)): # 세로 크기 (행 개수)
    for j in range(len(numbers[i])): # 가로 크기 (열 개수)
        print(numbers[i][j], end=' ')
    print()

print("-" * 20)

# 3. while문과 Unpacking
i = 0
while i < len(numbers): # 반복할 때 리스트의 크기 활용(세로 크기)
    x, y = numbers[i]
    print(x, y)
    i += 1

print("-" * 20)

# 4. 이중 while문 (인덱스로 접근하기)
i = 0
while i < len(numbers): # 세로 크기
    j = 0
    while j < len(numbers[i]): # 가로 크기
        print(numbers[i][j], end=' ')
        j += 1 # 가로 인덱스를 1 증가 시킴
    print()
    i += 1 # 세로 인덱스를 1 증가 시킴

4️⃣ 2차원 리스트 만들기: 건물 올리기 🏗️

빈 땅(빈 리스트)에 층을 하나씩 쌓아 올리는 방식입니다.

# 1부터 4까지 들어가는 2x2 리스트 만들기
grid = []

for i in range(2):
    row = []  # 새 층을 만듦
    for j in range(2):
        row.append(0) # 0으로 채움
    grid.append(row) # 완성된 층을 건물에 추가

print(grid)
# [[0, 0], [0, 0]]
✨ 고수의 팁: 리스트 표현식 (List Comprehension)
반복문을 한 줄로 줄여서 건물을 뚝딱 지을 수도 있습니다.
# 3층(행), 2호(열)짜리 건물 짓기
# 안쪽 [ ... ] : 호(Room) 만들기
# 바깥 [ ... ] : 층(Floor) 쌓기

grid = [[0 for j in range(2)] for i in range(3)]

print(grid)
# [[0, 0], [0, 0], [0, 0]]

5️⃣ 📝 건축가 자격 시험

1. `matrix = [[1, 2], [3, 4]]` 에서 `3`을 출력하려면?
  • a. matrix[0][0]
  • b. matrix[1][0]
  • c. matrix[0][1]
  • d. matrix[1][1]
2. [코딩하기] 빙고판 채우기

3x3 리스트를 만들고, 가운데(1행 1열)에만 "BINGO"라고 적어보세요. 나머지는 0입니다.

board = [
    [0, 0, 0],
    [0, 0, 0],
    [0, 0, 0]
]
board[__][__] = "BINGO"
print(board)
🔑 정답 확인하기 (클릭)
  1. b: 1번째 층(두번째 줄)의 0번째 방(첫번째 칸)입니다.
  2. 코딩 2: 1, 1 (board[1][1] = "BINGO")

📌 핵심 요약

  • 2차원 리스트: 리스트 안에 리스트가 있는 구조 (행/열).
  • 접근: 리스트[행][열] 로 두 번 인덱싱.
  • 출력: for문을 두 번 중첩해서 사용.
  • 데이터가 표(Grid) 형태일 때 필수적으로 사용됩니다!
