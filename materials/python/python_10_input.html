🔵 파이썬 마스터 코치: 10. 입력하기 (Input)

컴퓨터와 대화하는 첫 걸음! 내 말을 들어줘!

1️⃣ input() 함수: 마이크 켜기

지금까지는 우리가 코드를 미리 적어두고 실행만 했습니다. 이제는 컴퓨터가 우리의 대답을 기다리게 해봅시다.

🎤 마스터 코치의 비유: 노래방 마이크
input("질문 내용") 코드는 마이크를 켜는 것과 같습니다.
사용자가 말을 하고(입력), 엔터(Enter)를 칠 때까지 컴퓨터는 멈춰서 기다립니다.
입력한 내용은 문자열(String) 형태로 변수에 저장됩니다. 
# 1. 이름 입력받기
name = input("이름이 무엇입니까? ")

print("반갑습니다, " + name + "님!")

2️⃣ 자료형별 입력 방법: 의상 갈아입기

input()으로 받은 데이터는 무조건 문자(String)입니다. 숫자를 입력해도 문자로 취급되죠.

🎭 마스터 코치의 비유: 무대 의상 체인지
input()이라는 무대에서 내려오면 무조건 '문자 옷'을 입고 있습니다.
계산(더하기, 빼기 등)을 하려면 '숫자 옷'으로 갈아입혀야 합니다.
int()(정수), float()(소수) 탈의실을 이용하세요! 
# 나이 입력받기 (문자로 저장됨)
age = input("나이를 입력하세요: ") 
# print(age + 1) -> 에러! (문자와 숫자는 더할 수 없음)

# 정수로 변환하기 (의상 체인지!)
real_age = int(age)
print(real_age + 1) # 이제 더하기 가능!

# 실수(소수점) 입력받기
height = float(input("키를 입력하세요: "))
print("내년에는 " + str(height + 1.5) + "cm가 될 거예요!")

# 💡 한 줄로 줄여쓰기 (고수들의 방법)
score = int(input("점수를 입력하세요: "))

3️⃣ 공백을 두고 2개 이상 입력: 재료 썰기

"10 20" 처럼 한 줄에 여러 값을 입력받고 싶을 때가 있죠?

🔪 마스터 코치의 비유: 긴 김밥 썰기
input()은 한 줄을 통째로 문자열로 가져옵니다 ("10 20"이라는 긴 김밥).
.split()은 이 김밥을 공백(띄어쓰기)을 기준으로 싹뚝싹뚝 썰어줍니다.
썰린 조각들은 리스트(List)에 담기거나, 여러 변수에 나누어 담을 수 있습니다. 
# 공백을 기준으로 나누기
data = input("두 개의 숫자를 입력하세요 (예: 10 20): ").split()

print(data) # ['10', '20'] -> 문자열 리스트로 변신!

# 각각 변수에 담기 (언패킹)
a, b = input("이름과 나이를 띄어쓰기로 구분해 입력: ").split()
print("이름:", a)
print("나이:", b)

🔥 심화: 입력받아 즉시 숫자로 바꾸기 (Map)

split() 결과는 여전히 문자열입니다. map() 마법을 쓰면 한 번에 숫자로 바꿀 수 있어요!

# "10 20" -> [10, 20] (숫자로 변환!)
x, y = map(int, input("숫자 2개 입력: ").split())
print(x + y) # 바로 산술 연산 가능!

4️⃣ 📝 소통 능력 시험

1. `user = input()` 을 실행하고 '100'을 적었습니다. `type(user)`의 결과는?
  • a. <class 'int'> (정수)
  • b. <class 'str'> (문자열)
  • c. <class 'float'> (실수)
  • d. <class 'bool'> (불)
2. 사용자에게 나이를 입력받아 내년 나이를 출력하는 올바른 코드는?
  • a. print(input() + 1)
  • b. print(int(input()) + 1)
  • c. print(str(input()) + 1)
  • d. print(input(int) + 1)
3. [코딩하기] 두 수의 합 구하기

사용자로부터 두 개의 숫자(정수)를 공백으로 구분하여 입력받고, 두 수의 합을 출력하세요.

# 입력 예시: 10 20
a, b = map(_______, input("두 수 입력: ").split())

print(a + b) 
# 결과: 30
🔑 정답 확인하기 (클릭)
  1. b: input()은 무조건 문자열(str)을 반환합니다.
  2. b: 숫자로 계산하려면 int()로 변환해야 합니다.
  3. int: map(int, ...)를 써서 문자열을 정수로 변환합니다.

📌 핵심 요약

  • input(): 사용자 입력을 받는 함수. 결과는 항상 문자열(String)!
  • 형 변환: 숫자가 필요하면 int(), float()라는 탈의실을 거쳐야 함.
  • split(): input().split()으로 공백 기준 자르기.
  • map(): map(int, input().split())으로 여러 개를 한 번에 숫자로 변신!
