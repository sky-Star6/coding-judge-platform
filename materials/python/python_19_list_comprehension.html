🟣 파이썬 마스터 코치: 19. 리스트 표현식

파이썬의 '꽃'이라 불리는 마법 같은 한 줄 코딩! ✨

1️⃣ 기본 리스트 표현식: 마법 공장 🏭

반복해서 리스트를 만드는 과정을 단 한 줄로 줄여줍니다. (List Comprehension)

🏭 마스터 코치의 비유: 마법 공장
재료(기존 리스트)를 넣으면, 기계(표현식)가 자동으로 가공해서
완성품(새 리스트)을 툭툭 뱉어냅니다.

VS 비교 체험

# [옛날 방식] for문으로 하나씩 추가하기
nums = [1, 2, 3, 4]
result = []
for n in nums:
    result.append(n * 3)
print(result) # [3, 6, 9, 12]

# [마법 방식] 리스트 표현식 (추천!) ✨
# 해석: nums에 있는 n을 하나씩 꺼내서, n*3 한 것을 리스트로 만들어라!
result = [n * 3 for n in nums]
print(result) # [3, 6, 9, 12]

2️⃣ 조건이 있는 표현식: 품질 관리 🕵️‍♂️

원하는 데이터만 쏙쏙 골라서(if) 가공할 수 있습니다.

🕵️‍♂️ 마스터 코치의 비유: 필터
공장 레일 위에 '품질 관리자'가 서 있습니다.
"짝수만 통과!", "5보다 큰 것만 통과!"라고 외치며 걸러냅니다. 
# 1부터 10까지 중 '짝수'만 골라서 제곱하기
# 문법: [표현식 for 변수 in 리스트 if 조건]

evens = [x ** 2 for x in range(1, 11) if x % 2 == 0]
print(evens)

# 출력 결과: [4, 16, 36, 64, 100]
# (2, 4, 6, 8, 10의 제곱들)

3️⃣ 중첩 표현식: 이중 공정 🍰

반복문이 두 개 이상일 때도 한 줄로 쓸 수 있습니다. (구구단 등)

# 구구단 2단, 3단 만들기
# 앞에 있는 for문이 바깥쪽 루프입니다.

gugu = [x * y for x in range(2, 4) for y in range(1, 10)]
print(gugu)

# 출력: [2, 4, 6... 18, 3, 6, 9... 27]

4️⃣ map() 함수: 변신 레이저 🔫

리스트의 모든 요소를 함수에 통과시켜서 변환합니다.

🔫 마스터 코치의 비유: 변신 레이저
리스트 전체에 레이저를 쏘면,
모든 아이템이 지정된 모양(함수)으로 변신합니다. 
nums = ['1', '2', '3']

# 모든 문자를 정수(int)로 변환!
# map(변환함수, 리스트)
items = list(map(int, nums))

print(items)       # [1, 2, 3]
print(sum(items))  # 6 (숫자니까 더하기 가능!)

5️⃣ 📝 마법사 승급 시험

1. 다음 중 '홀수'만 골라내는 올바른 리스트 표현식은?
  • a. [n for n in range(10) if n % 2 == 1]
  • b. [n if n % 2 == 1 for n in range(10)]
  • c. [for n in range(10) return n % 2]
  • d. [n ** 2 for n in range(10)]
2. [코딩하기] 양수만 제곱하기

리스트 nums = [1, -2, 3, -4, 5]에서
양수(0보다 큰 수)만 골라서 그 수를 제곱한 리스트를 만드세요. 

nums = [1, -2, 3, -4, 5]
result = [_______ for n in nums if _______]
print(result)
🔑 정답 확인하기 (클릭)
  1. a: if 조건문은 뒤에 위치합니다.
  2. 코딩 2: n ** 2, n > 0 (양수만 골라 제곱)

📌 핵심 요약

  • 리스트 표현식: [표현식 for 변수 in 리스트] 형태로 한 줄 요약!
  • 조건 필터링: 뒤에 if 조건을 붙여서 원하는 것만 쏙!
  • map(): 모든 요소에 함수를 적용할 때 사용 (list()로 변환 필요).
