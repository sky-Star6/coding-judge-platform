💖 파이썬 마스터 코치: 5. 문자열 이야기

글자들을 예쁘게 꿰어 보석같은 문장을 만들어봐요!

1️⃣ 문자열(String) 이란?

프로그래밍에서는 글자들의 묶음을 문자열이라고 부릅니다.

📿 마스터 코치의 비유: 구슬 팔찌와 사탕 포장지
문자열은 글자(구슬)를 꿰어 만든 팔찌같아요.
그리고 이 팔찌를 컴퓨터에게 줄 때는 꼭 ' ' (작은 따옴표)나 " " (큰 따옴표)라는 사탕 포장지로 감싸야 해요.
안 그러면 컴퓨터는 이게 글자인지, 명령어인지 헷갈려하거든요!

📦 포장하는 방법

print('마이로봇')  # 작은 따옴표 포장
print("안녕")      # 큰 따옴표 포장

# 따옴표를 안 쓰면?
print(안녕) # 에러 발생! (안녕이라는 변수나 함수를 찾으려고 함)

🧐 포장지 안에 따옴표를 넣고 싶다면?

서로 다른 종류의 따옴표를 사용하면 됩니다.

print("'파이썬' 재밌다") # 밖에 큰 따옴표 사용
print('"안녕"이라고 말했다') # 밖에 작은 따옴표 사용

2️⃣ 여러 줄 문자열 (Multi-line String)

줄바꿈이 있는 긴 문장을 적고 싶을 때는 어떻게 할까요?

📜 마스터 코치의 비유: 고대의 긴 두루마리
짧은 문장은 따옴표 하나지만, 아주 긴 두루마리 같은 문장은
따옴표 3개(''' 또는 """)를 연속으로 찍어서 시작하고 끝내야 합니다.

💻 코드 예제

# 따옴표 3개로 감싸기
long_text = '''마이로봇은
너무 즐거워
마이로봇은
너무 재밌어'''

print(long_text)
# 출력 결과가 줄바꿈 그대로 나옵니다!

3️⃣ 문자열 연산 (더하기, 곱하기)

글자끼리 더하고 곱한다고? 파이썬에서는 가능합니다!

🔗 마스터 코치의 비유: 연결 고리와 복사 도장
+ (더하기): 두 단어를 강력 접착제로 딱 붙이는 것!
* (곱하기): 똑같은 내용을 여러 번 찍어내는 복사 도장(Stamp)!

➕ 문자열 연결하기 (Concatenation)

print('마이로봇' + '안녕')        
# 출력: 마이로봇안녕 (딱 붙음)

print('마이로봇 ' + '안녕')       
# 출력: 마이로봇 안녕 (공백도 글자니까 포함해야 띄어짐)

# ⚠️ 주의: 숫자랑 더할 때는 꼭 문자로 바꿔주기!
# print('나이: ' + 10)  -> 에러!
print('나이: ' + str(10)) # 출력: 나이: 10

✖️ 문자열 반복하기 (Repetition)

print('마이로봇' * 3)
# 출력: 마이로봇마이로봇마이로봇 (3번 반복)

print('반짝' * 2)
# 출력: 반짝반짝

4️⃣ 📝 문자열 퀴즈

1. 다음 중 에러가 발생하는 코드는?
  • a. print("Hello")
  • b. print('Hello')
  • c. print(Hello)
  • d. print("Hello " + "World")
2. 여러 줄의 문자열을 작성할 때 사용하는 기호는?
  • a. " (한 개)
  • b. ' (한 개)
  • c. ''' (세 개)
  • d. # (샵)
3. 다음 코드의 실행 결과는? 
print("7" + 7)
  • a. 14
  • b. 77
  • c. 49
  • d. 에러 발생
4. "Python"이란 글자가 3번 반복되게 하려면?
  • a. "Python" + 3
  • b. "Python" * 3
  • c. "Python" ^ 3
  • d. 3("Python")
5. [코딩하기] 다음 실행 결과가 나오도록 빈칸을 채워주세요. (반복과 연결 사용) 
# 조건: '★'을 5번 출력하고, 그 뒤에 '슈퍼스타'를 붙여주세요.
print("★" * ____ + "_______")

실행 결과: ★★★★★슈퍼스타

🔑 정답 확인하기 (클릭)
  1. c: 따옴표가 없어서 변수 Hello를 찾다가 실패합니다.
  2. c: 따옴표 3개(''' 또는 """)를 사용합니다.
  3. d: 문자열("7")과 숫자(7)는 직접 더할 수 없습니다. str(7)해야 합니다.
  4. b: 문자열 반복은 곱하기(*) 연산자를 사용합니다.
  5. 5, 슈퍼스타: print("★" * 5 + "슈퍼스타")

📌 핵심 요약

  • 문자열: 따옴표(' 또는 ")로 감싸야 함.
  • 여러 줄: 따옴표 3개(''')로 감쌈.
  • 연산: +는 연결, *는 반복.
  • 주의: 숫자와 문자는 그냥 더할 수 없다! str(숫자)로 변신시키기.
