💗 파이썬 마스터 코치: 22. 회문 판별하기

거꾸로 읽어도 똑같은 문장을 찾아봅시다! (Palindrome)

1️⃣ 회문(Palindrome)이란? 데칼코마니 🦋

앞에서 읽으나 뒤에서 읽으나 똑같은 단어나 문장을 말합니다.

LEVEL ↔️ LEVEL
"기러기", "토마토", "스위스", "내 이름은 우영우"
"level", "radar", "noon"
🦋 마스터 코치의 비유: 데칼코마니
종이를 반으로 접었을 때, 양쪽이 완벽하게 겹치는 그림과 같습니다.
(왼쪽 끝 == 오른쪽 끝)

2️⃣ for문으로 판별하기: 정밀 수사 🕵️‍♀️

양쪽 끝에서 시작해서 가운데로 오면서 글자가 같은지 하나씩 확인합니다.

🕵️‍♀️ 마스터 코치의 비유: 양쪽 수사관
왼쪽 수사관(0번)과 오른쪽 수사관(-1번)이 글자를 비교합니다.
둘이 다르면? -> "범인이다!(회문 아님)" 라고 외치고 수사 종료(break).
끝까지 다르면 -> "무죄!(회문 맞음)" 
word = "level"
is_palindrome = True # 일단 맞다고 가정

# 절반까지만 검사하면 됨 (len(word) // 2)
for i in range(len(word) // 2):
    # 왼쪽 글자 vs 오른쪽 글자
    if word[i] != word[-1 - i]:
        is_palindrome = False # 하나라도 다르면 탈락!
        break

print(is_palindrome) # True

3️⃣ 슬라이싱으로 판별하기: 마법 거울 ✨

파이썬의 슬라이싱([::-1])을 쓰면 한 방에 뒤집을 수 있습니다. 가장 파이썬다운 방법(Pythonic)입니다.

✨ 마스터 코치의 비유: 마법 거울
복잡하게 비교할 필요 없이, 거울에 비친 모습과 원래 모습이
"똑같니?" (==) 라고 물어보면 끝! 
word = "radar"

# 뒤집은 단어 만들기
reversed_word = word[::-1]

# 원래 단어와 비교
if word == reversed_word:
    print("회문입니다!")
else:
    print("회문이 아닙니다.")
