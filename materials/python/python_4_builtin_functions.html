🛠️ 파이썬 마스터 코치: 4. 내장 함수

파이썬이 미리 준비해 둔 강력한 도구들을 꺼내 써볼까요?

1️⃣ 내장 함수 (Built-in Function) 란?

파이썬을 설치하자마자 바로 사용할 수 있도록 미리 만들어진 함수들입니다.

🧰 마스터 코치의 비유: 마스터의 공구함
우리가 집을 지을 때 망치, 톱, 드라이버를 직접 만들어서 쓰진 않죠?
이미 공구함에 들어있는 도구(내장 함수)를 꺼내서 쓰기만 하면 됩니다! 편리하죠?

📦 내 도구함 구경하기: dir(__builtins__)

내 공구함에 무슨 도구가 있는지 궁금하다면 이 주문을 외워보세요.

print(dir(__builtins__))
# 출력: ['abs', 'all', 'any', ..., 'print', 'len', 'sum', ...]
# 엄청나게 많은 도구들이 쏟아져 나올 거예요!

2️⃣ 자주 쓰는 핵심 도구들

📏 길이와 크기 재기 (len, max, min, sum)

데이터들을 측정할 때 사용하는 도구들입니다.

함수 비유 (Metaphor) 설명
len() 📏 줄자 길이를 잽니다. (리스트 개수, 문자열 길이 등)
max() 🥇 금메달리스트 가장 큰 녀석을 찾아냅니다.
min() 👶 막내 찾기 가장 작은 녀석을 찾아냅니다.
sum() 💰 저금통 털기 모든 숫자를 다 더합니다. 
nums = [10, 20, 30, 5]

print(len(nums))  # 출력: 4 (개수가 4개)
print(max(nums))  # 출력: 30 (제일 큰 형님)
print(min(nums))  # 출력: 5 (제일 작은 막내)
print(sum(nums))  # 출력: 65 (다 더한 값)

🪞 수학 도구와 변환 도구 (abs, int, str...)

👓 마스터 코치의 비유: 번역 안경
int(), str() 같은 함수는 데이터를 다른 형태로 바꿔주는 안경과 같아요.
bin(), hex()는 숫자를 컴퓨터 언어(2진수)나 16진수로 보여주는 특수 안경이죠!
  • abs(x): 절댓값 (음수를 양수로, 긍정의 거울 🪞)
  • int(x): 정수로 변환 (소수점 자르기 ✂️)
  • float(x): 실수로 변환 (소수점 붙이기 💧)
  • str(x): 문자로 변환 (따옴표 붙이기 🏷️)
  • bin(x): 2진수로 변환 (0b...)
  • oct(x): 8진수로 변환 (0o...)
  • hex(x): 16진수로 변환 (0x...)
print(abs(-100))    # 출력: 100
print(int(3.14))    # 출력: 3
print(float(3))     # 출력: 3.0
print(bin(10))      # 출력: 0b1010
print(oct(10))      # 출력: 0o12
print(hex(255))     # 출력: 0xff

3️⃣ 📝 내장 함수 실력 점검

1. 다음 리스트의 길이를 구하기 위해 빈칸에 들어갈 알맞은 함수는? 
fruit = ['apple', 'banana', 'cherry']
print(______(fruit)) # 출력 결과: 3
  • a. len()
  • b. length()
  • c. size()
  • d. count()
2. print(type('100'))의 실행 결과는 무엇인가요?
  • a. <class 'int'>
  • b. <class 'str'>
  • c. <class 'float'>
  • d. <class 'list'>
3. 수직선 위에서 0으로부터의 거리(절댓값)를 구하는 함수는? 
print(____(-5)) # 출력 결과: 5
  • a. sum()
  • b. abs()
  • c. max()
  • d. min()
4. 빈칸에 알맞은 코드를 채워주세요. (주관식) 
print(___(45, 22, 56))        # 가장 큰 값 (56)
print(___([10, 20]))          # 합계 (30)
🔑 정답 확인하기 (클릭)
  1. a. len(): 길이는 length의 줄임말인 len을 씁니다.
  2. b. <class 'str'>: 따옴표가 있으므로 문자열(string)입니다.
  3. b. abs(): absolute value(절댓값)의 줄임말입니다.
  4. max, sum: 가장 큰 값은 max(), 합계는 sum() 입니다.

📌 핵심 요약

  • 내장 함수: 파이썬이 공짜로 주는 도구 상자(Toolbox) 🧰
  • 측정 도구: len(길이), max(최대), min(최소), sum(합계)
  • 변환 도구: int, float, str, abs(절댓값), bin(2진수), oct(8진수), hex(16진수)
