🟣 파이썬 마스터 코치: 13. 조건문 (if)

인생은 선택의 연속! 컴퓨터에게 똑똑한 선택을 가르쳐봅시다.

1️⃣ IF 문: 만약에 말이야... 🚦

조건문은 "만약 ~라면, 이것을 실행해!"라고 컴퓨터에게 명령하는 것입니다.

🚦 마스터 코치의 비유: 신호등
조건식: "초록불인가?" (True/False)
True: 건너간다 (실행)
False: 멈춰 있는다 (실행 안 하고 건너뜀)

🚨 매우 중요: 들여쓰기 (Indentation)

파이썬은 들여쓰기(Space 4칸 또는 Tab 1번)로 "어디까지가 if문 안쪽인지"를 판단합니다.
들여쓰기가 된 부분은 마치 "VIP 구역" 🎫과 같아서, 조건이 맞을 때만 들어갈 수 있습니다.

money = 5000

if money >= 3000: # 조건식 끝에 콜론(:) 필수!
    # 여기부터 VIP 구역 (들여쓰기)
    print("택시를 타고 가라") 
    print("편안하게 도착!")
    # VIP 구역 끝

print("집 도착") # 들여쓰기 없음 (항상 실행)

2️⃣ ELSE 문: 그렇지 않다면... 🛡️

조건이 맞지 않을 때(False) 실행할 "플랜 B"를 준비합니다.

🛡️ 마스터 코치의 비유: 양자택일
동전 던지기처럼 앞면(if)이 아니면 무조건 뒷면(else)입니다.
else에는 조건을 적지 않습니다! (나머지 전부니까요) 
weather = "비"

if weather == "맑음":
    print("우산을 두고 가세요") # 맑을 때만
else:
    print("우산을 챙기세요")     # 맑지 않은 모든 경우 (비, 눈, 흐림...)

3️⃣ ELIF 문: 그게 아니고 이거라면... 📋

선택지가 여러 개일 때(다지선다) 사용합니다. else if를 줄인 말입니다.

📋 마스터 코치의 비유: 메뉴 고르기
1. 스테이크 있나요? (if) -> 네, 주세요. (끝)
2. (없으면) 그럼 파스타는요? (elif) -> 네, 주세요. (끝)
3. (없으면) 그럼 그냥 물만 주세요. (else) 
score = 85

if score >= 90:
    print("성적: A")
elif score >= 80:
    print("성적: B") # 90은 안 넘지만 80은 넘을 때
elif score >= 70:
    print("성적: C")
else:
    print("성적: F") # 70점 미만인 모든 경우
⚙️ 동작 방식 (중요!)
위에서부터 차례대로 조건을 검사합니다.
하나라도 참(True)이 나와서 실행되면, 나머지 아래쪽(elif, else)은 쳐다보지도 않고 건너뜁니다!

🔍 If 조건문의 동작 방식 알아보기

프로그램이 if문을 만났을 때, 머릿속으로 이렇게 생각합니다:

  1. "조건식이 참(True)인가?" 하고 물어봅니다.
  2. [YES] 참이라면: 안쪽(들여쓰기 된 부분)으로 들어가서 코드를 실행합니다.
  3. [NO] 거짓이라면: 안쪽 코드를 무시하고(Skip) 바로 if문 다음 줄로 넘어갑니다.
조건 확인 (True?) ➡️ [YES] 실행 / [NO] 건너뜀 ➡️ 다음 줄로 이동

🕵️‍♀️ 0은 거짓말쟁이? (참과 거짓의 비밀)

파이썬에서는 True, False라고 딱 잘라 말하지 않아도, 참과 거짓으로 판단하는 값들이 있습니다.

  • 거짓(False) 취급: 0, None, 비어있는 문자열("")
  • 참(True) 취급: 0이 아닌 숫자(1, -5, 12.5), 내용이 있는 문자열("Hello")
# 1. 거짓(False)인 친구들 (실행 안 됨)
if 0: print("0은 거짓")
if None: print("None도 거짓")
if "": print("빈 문자열도 거짓")

# 2. 참(True)인 친구들 (실행 됨!)
if 1: print("1은 참")
if 12.5: print("실수도 참")
if "Hello": print("글자가 있으면 참")
if 0x1F: print("16진수도 0 아니면 참")

4️⃣ 다중 조건과 중첩: 깐깐한 조건 🧐

여러 조건을 한 번에 검사하거나, 조건 안에 또 조건을 넣을 수 있습니다.

논리 연산자 (and, or, not)

  • and (그리고): 둘 다 참이어야 통과 (엄격함)
  • or (또는): 둘 중 하나만 참이면 통과 (관대함)
age = 20
has_ticket = True

# 나이도 19세 이상이고(and), 티켓도 있어야 함
if age >= 19 and has_ticket:
    print("입장 가능합니다!")
else:
    print("입장 불가!")

5️⃣ 📝 논리력 테스트

1. 다음 코드의 실행 결과는? 
x = 10
if x > 5:
    print("A")
elif x > 8:
    print("B")
else:
    print("C")
  • a. A
  • b. B
  • c. A 그리고 B
  • d. C
2. 들여쓰기(Indentation)에 대한 설명으로 틀린 것은?
  • a. if문 조건이 참일 때 실행할 코드를 들여씁니다.
  • b. 보통 스페이스 4칸을 사용합니다.
  • c. 들여쓰기를 안 해도 프로그램은 잘 동작합니다.
  • d. 같은 블록 내에서는 들여쓰기 칸 수가 같아야 합니다.
3. [코딩하기] 놀이공원 입장료 계산

8세 미만은 무료, 20세 미만은 학생 요금, 60세 이상은 우대 요금입니다.

age = 30

if age < 8:
    print("무료")
____ age < 20: 
    print("학생 요금")
____ age >= 60:
    print("경로 우대")
____:
    print("성인 요금")
    
# 결과: 성인 요금
🔑 정답 확인하기 (클릭)
  1. a: x > 5가 참이므로 "A" 출력 후 나머지는 건너뜁니다! (B는 실행 안 됨)
  2. c: 파이썬에서 들여쓰기는 문법 그 자체입니다. 안 하면 에러나요! 🚨
  3. elif, elif, else: 순서대로 조건을 검사해야 합니다.

📌 핵심 요약

  • if: 조건이 참일 때만 실행하는 "VIP 구역"을 만듭니다.
  • 들여쓰기: VIP 구역을 표시하는 절대적인 규칙입니다. (필수!)
  • else: 모든 조건이 거짓일 때 실행할 최후의 보루입니다.
  • elif: 여러 조건을 순서대로 검사할 때 사용합니다.
  • 동작: 위에서 아래로 흐르며, 한번 걸리면 나머지는 통과(Skip)합니다.
