🐤 파이썬 마스터 코치: 시작하기

키워드, 식별자, 자료형, 출력하기 완벽 정복!

1️⃣ 키워드 (Keyword)

파이썬이 미리 찜해둔 특별한 단어들입니다. 이 단어들은 파이썬 문법에서 특별한 기능을 수행하기 때문에, 우리가 마음대로 이름을 붙여서 사용할 수 없습니다.

🎟️ 마스터 코치의 비유: VIP 예약석
영화관의 VIP 예약석은 이미 주인이 정해져 있어서 우리가 앉을 수 없죠? 키워드도 마찬가지입니다. 파이썬이 미리 예약해둔 자리라서 변수 이름으로 쓸 수 없어요!

🚫 예약된 키워드 목록 (사용 금지!)

False, None, True, and, as, assert, 
break, class, continue, def, del, elif, 
else, except, finally, for, from, global, 
if, import, in, is, lambda, nonlocal, 
not, or, pass, raise, return, try, 
while, with, yield
⚠️ 주의: 대소문자를 구분해요!
파이썬은 대문자와 소문자를 엄격하게 구분합니다.
예를 들어 True는 예약어(키워드)이지만, true는 예약어가 아닙니다. (키워드가 아니므로 변수명으로 사용 가능하지만, 헷갈리니 권장하진 않아요!)

2️⃣ 식별자 & 이름 짓기 (Naming)

우리가 변수, 함수, 클래스 등에 붙여주는 이름을 식별자라고 합니다. 식별자를 지을 때는 꼭 지켜야 할 규칙과 암묵적인 스타일이 있어요!

🚫 식별자 절대 규칙 (안 지키면 에러!)

  • 가능한 문자: 영문자(대소문자), 숫자(0~9), 밑줄(_)만 사용할 수 있어요.
  • 숫자로 시작 금지: 1st_name (X) / name_1 (O)
  • 공백 금지: 빈칸은 쓸 수 없어요. 대신 밑줄(_)을 활용하세요. my score (X) / my_score (O)
  • 키워드 사용 금지: 파이썬 예약어(if, for, with 등)는 쓸 수 없어요.
🏷️ 마스터 코치의 비유: 스타일이 다른 명찰
친구끼리는 편한 별명(Snake), 격식 있는 자리엔 정장(Pascal)을 입 듯, 역할에 따라 이름 스타일이 달라요.

🐍 스네이크 케이스 (Snake Case)

  • 변수(Variable)와 함수(Function) 이름에 사용합니다.
  • 단어 사이를 언더바(_)로 연결합니다. (뱀처럼 바닥을 기어가는 모양!)
  • 예: my_score, print_hello()

🐫 파스칼 케이스 (Pascal Case)

  • 클래스(Class) 이름에 사용합니다.
  • 각 단어의 첫 글자를 대문자로 씁니다. (언더바 없음)
  • 예: MyCar, StudentProfile

3️⃣ 자료형과 출력 (Data Types & Print)

📊 기본 자료형 3총사

  • int (정수): 소수점 없는 숫자 (예: 나이, 개수)
    age = 20
  • float (실수): 소수점 있는 숫자 (예: 키, 몸무게)
    height = 175.5
  • str (문자열): 글자. 따옴표로 감싸야 함 (예: 이름)
    name = "Alice"

🤔 123 vs '123' 무엇이 다를까?

출력 결과는 같아 보이지만, 컴퓨터는 완전히 다르게 인식합니다!

🔢 숫자 123 (int): 계산기 두드릴 때 쓰는 진짜 숫자 (더하기 가능)
🔤 문자 '123' (str): 그냥 글자로 적힌 그림 (더하기 불가능) 
print(123)   # 숫자 (int) -> 연산 가능
print('123') # 문자 (str) -> 연산 불가 (텍스트일 뿐)

print(100 + 100)     # 결과: 200 (덧셈)
print('100' + '100') # 결과: 100100 (글자 이어붙이기)

4️⃣ 이스케이프 시퀀스 & Raw String

문자열 안에서 백슬래시(\)를 만나면 특별한 일이 일어납니다!

🏃 이스케이프 시퀀스 (Escape Sequence)

백슬래시(\)와 특정 문자를 조합해 특별한 기능을 수행합니다.

  • \n : 줄바꿈 (Enter)
  • \t : 탭 간격 띄우기 (Tab)
  • \', \" : 따옴표 자체를 출력하고 싶을 때
  • \\ : 백슬래시 자체를 출력하고 싶을 때
print("Hello\nWorld") 
# 출력:
# Hello
# World

print("Hello\tWorld")
# 출력: Hello    World

🥩 날것 그대로! Raw String

문자열 앞에 r을 붙이면 백슬래시를 기능이 아닌 글자 그대로 인식합니다. (주로 파일 경로나 정규표현식에 사용)

print(r"C:\Users\Name") 
# 출력: C:\Users\Name (\n을 줄바꿈으로 인식 안 함!)

5️⃣ 주석 (Comment)

프로그램 실행에는 전혀 영향을 주지 않고, 사람(개발자)을 위해 남겨두는 메모입니다.

📝 마스터 코치의 비유: 투명 포스트잇 (비밀 쪽지)
선생님(컴퓨터) 눈에는 안 보이고, 우리(사람) 눈에만 보이는 비밀 메모에요.
어려운 코드에 설명을 달거나, 잠시 코드를 꺼두고 싶을 때 사용해요.

🛠️ 사용하는 방법

한 줄 주석: # 기호를 사용합니다.

# 이 줄은 컴퓨터가 무시합니다.
print("Hello") # 코드 뒤에 설명을 적을 수도 있어요.

여러 줄 주석: 따옴표 3개(''' 또는 """)를 사용합니다.

'''
여기는 여러 줄을
자유롭게 쓸 수 있는
메모장 공간입니다.
'''

1. 다음 중 파이썬의 예약어(Keyword)가 아닌 것을 고르세요.
  • a. if
  • b. while
  • c. print
  • d. True
  • e. import
2. 다음 중 'Hello, world!' 를 출력하는 방법으로 올바른 것을 고르세요.
  • a. print 'Hello, world!'
  • b. print Hello, world!
  • c. print(Hello, world!)
  • d. print('Hello, world!')
  • e. print['Hello, world!']
3. 다음 소스 코드를 완성하여 각 줄에 출력되게 만드세요. 
print('Hello, world!')
print(     )  # 빈칸

실행결과:
Hello, world!
1234x7890

4. 다음 중 출력 결과가 다른 하나를 고르세요.
  • a. print("Hello\nWorld")
  • b. print("Hello", "World", sep="\n")
  • c. print("""Hello World""")
  • d. print("Hello\tWorld")
🔑 정답 확인하기 (클릭)
  1. c. print: print는 내장 함수입니다. (예약어 아님)
  2. d. print('Hello, world!'): 문자열 출력 시 따옴표와 괄호가 필수입니다.
  3. '1234x7890': 문자가 포함되어 있으므로 따옴표를 사용하여 문자열로 만들어야 합니다.
  4. c. print("""Hello World"""): 이것만 줄바꿈 없이 한 줄로 출력됩니다. (a, b는 줄바꿈, d는 탭으로 띄움)

📌 핵심 요약

  • 식별자: 변수(snake_case), 클래스(PascalCase)
  • 자료형: int(정수), float(실수), str(문자열)
  • 이스케이프: \n(엔터), \t(탭), r""(기능무시)
  • 출력: 숫자와 문자는 다르다! (연산 가능 여부 차이)
