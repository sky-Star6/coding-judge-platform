🟡 파이썬 마스터 코치: 23. N-gram

문자나 단어를 N개씩 묶어서 살펴봅시다! 연관 검색어의 비밀! 🔗

1️⃣ N-gram이란? 이동하는 창문 🪟

문자열이나 리스트에서 N개의 연속된 아이템을 묶어서 자르는 방법입니다. 검색 엔진이나 AI가 다음 단어를 예측할 때 주로 사용합니다.

H e l l o
▲ 2-gram (Bigram): 두 글자씩 묶기 ("He")
🔗 마스터 코치의 비유: 기차 놀이
친구 두 명이 손을 잡고 걸어갑니다. (2-gram)
한 칸씩 옆으로 이동하면서 "너랑 나랑 짝꿍!" 하고 외치는 것과 같습니다.
("He", "el", "ll", "lo")

2️⃣ 문자 N-gram: 글자끼리 묶기 🅰️🅱️

문자열 안에서 글자를 N개씩 묶어서 출력합니다.

text = "Hello"
N = 2 # 2개씩 묶기 (Bigram)

# 중요: 전체 길이에서 (N-1) 만큼 뺀 곳까지만 반복해야 함!
# 아니면 인덱스 에러(IndexError)가 발생합니다.
for i in range(len(text) - N + 1):
    # 현재 위치(i)부터 N개만큼 슬라이싱
    print(text[i : i + N])

# 출력 결과:
# He
# el
# ll
# lo

3️⃣ 단어 N-gram: 단어끼리 묶기 🚃🚃

문장이 주어졌을 때, 단어(공백 기준)를 N개씩 묶습니다.

sentence = "There is no spoon"
words = sentence.split() # ['There', 'is', 'no', 'spoon']
N = 2

for i in range(len(words) - N + 1):
    print(words[i], words[i + 1])

# 출력 결과:
# There is
# is no
# no spoon

4️⃣ ✨ 고수의 팁: zip()으로 한 방에! 🤐

zip() 함수를 쓰면 인덱스 계산 없이 우아하게 해결할 수 있습니다.

🤐 마스터 코치의 비유: 지퍼(Zipper)
원본(text)과 한 칸 밀린 복사본(text[1:])을
지퍼처럼 드르륵~ 잠그면(zip) 2-gram이 완성됩니다! 
text = "hello"

# text      : ['h', 'e', 'l', 'l', 'o']
# text[1:]  :      ['e', 'l', 'l', 'o']
# -------------------------------------
# zip 결과  : ('h','e'), ('e','l'), ...

for a, b in zip(text, text[1:]):
    print(a, b)

📌 핵심 요약

  • N-gram: N개의 연속된 아이템을 묶는 것.
  • 반복 범위: range(len(데이터) - N + 1) (이 공식이 핵심!).
  • 문자열: 글자 단위, 리스트: 단어 단위로 적용 가능.
  • zip(data, data[1:])을 쓰면 더 편하게 할 수 있다!
