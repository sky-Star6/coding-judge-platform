🎀 파이썬 마스터 코치: 11. 문자열 가공과 조작

글자를 자르고, 붙이고, 꾸미는 텍스트 마법사가 되어보세요!

1️⃣ 길이와 개수 세기: 줄자와 계수기

가장 기본이 되는 것은 '얼마나 긴지' 그리고 '몇 개가 있는지' 아는 것입니다.

📏 마스터 코치의 비유: 줄자와 계수기
len()은 줄자입니다. 전체 길이를 잽니다.
.count()는 계수기(Clicker)입니다. 특정 글자가 몇 번 나오는지 셉니다. 
text = "Banana"

# 1. 길이 재기 (Length)
print(len(text))      # 6 (글자 수)

# 2. 개수 세기 (Count)
print(text.count('a')) # 3 ('a'가 3개)
print(text.count('na')) # 2 ('na'가 2번 등장: Ba-na-na)

2️⃣ 위치 찾기: 보물 지도 찾기

원하는 글자가 "몇 번째"에 있는지 찾아줍니다.

📍 마스터 코치의 비유: 네비게이션
.find()와 .index()는 둘 다 위치(인덱스)를 알려줍니다.
차이점: 길을 못 찾았을 때(글자가 없을 때)
- find: "못 찾겠어요(-1)"라고 침착하게 말합니다.
- index: "으악! 없어요!(Error)"라고 비명을 지릅니다. 
s = "Python"

# 위치 찾기
print(s.find('y'))   # 1 (1번 인덱스에 있음)
print(s.index('y'))  # 1

# 없을 때 반응 차이
print(s.find('z'))   # -1 (안전)
# print(s.index('z')) # ValueError: substring not found (에러 발생!)

🔙 뒤에서부터 찾기 (Right/Reverse)

함수 이름 앞에 'r'이 붙으면 오른쪽(Right) 끝에서부터 찾기 시작합니다.

  • rfind(): 뒤에서부터 찾아 인덱스 반환
  • rindex(): 뒤에서부터 찾아 인덱스 반환 (없으면 에러)
filename = "photo.jpg.bak"

# 앞에서 찾기 (find)
print(filename.find('.'))   # 5 (첫 번째 점)

# 뒤에서 찾기 (rfind) - 확장자 찾을 때 유용!
print(filename.rfind('.'))  # 9 (마지막 점)

3️⃣ 변신과 꾸미기: 메이크업 룸

문자열의 형태를 바꾸거나 내용을 교체합니다.

💄 마스터 코치의 비유: 스타일링
upper(): 하이힐을 신은 것처럼 키가 커집니다(대문자).
lower(): 단화를 신은 것처럼 작아집니다(소문자).
replace(): 낡은 옷을 새 옷으로 갈아입습니다(대체). 
msg = "Hello World"

print(msg.upper())    # "HELLO WORLD" (대문자로)
print(msg.lower())    # "hello world" (소문자로)
print(msg.swapcase()) # "hELLO wORLD" (대소문자 반대로)

# 바꾸기 (Old -> New)
print(msg.replace("World", "Python")) 
# "Hello Python"

4️⃣ 자르고 붙이기: 편집의 기술

split()의 반대 기능인 join()을 알아봅시다.

🔗 마스터 코치의 비유: 풀칠하기
split()이 가위라면, join()은 풀(Glue)입니다.
떨어져 있는 글자 조각(리스트)들을 풀("-", "," 등)로 이어 붙여서 하나의 긴 문장으로 만듭니다. 
words = "010 1234 5678"

# 공백으로 자르기 (Split)
print(words.split())  # ['010', '1234', '5678']

# 특정 문자로 자르기
date = "2025-12-25"
print(date.split("-")) # ['2025', '12', '25']

# ----------------------------------------

phone_parts = ["010", "1234", "5678"]

# 대시(-)로 이어 붙이기
phone = "-".join(words)
print(phone)  # "010-1234-5678"

# 그냥 이어 붙이기 (빈 문자열 사용)
text = "".join(["안", "녕"])
print(text)   # "안녕"

5️⃣ 청소와 정리: 공백 제거

글자 앞뒤에 붙은 불필요한 공백(띄어쓰기, 줄바꿈)을 깨끗하게 지웁니다.

🧹 마스터 코치의 비유: 먼지 털기
lstrip(): 왼쪽(Left) 먼지만 텁니다.
rstrip(): 오른쪽(Right) 먼지만 텁니다.
strip(): 양쪽을 싹싹 닦아냅니다 (가장 많이 씀!). 
dirty = "   Python   "

print(dirty.lstrip()) # "Python   " (왼쪽만)
print(dirty.rstrip()) # "   Python" (오른쪽만)
print(dirty.strip())  # "Python"    (양쪽 다!)

# 특정 문자 지우기 (콤마, 마침표)
a = ", Python."
print(a.lstrip(',.')) # " Python." (왼쪽 ,. 제거)
print(a.rstrip(',.')) # ", Python" (오른쪽 ,. 제거)
print(a.strip(',.'))  # " Python"  (양쪽 ,. 제거)

6️⃣ 검사하기: 탐정의 눈

문자열이 어떤 구성인지 확인하여 참(True) 또는 거짓(False)을 알려줍니다.

🕵️ 마스터 코치의 비유: 성분 분석기
"숫자로만 되어 있나?", "알파벳으로만 되어 있나?"를 조사합니다.
함수 이름이 is로 시작하면 "그렇습니까?"라고 묻는 것입니다. 
# 숫자(Digit)로만 되어 있는지?
print("1234".isdigit())   # True
print("123a".isdigit())   # False

# 알파벳(Alpha)으로만 되어 있는지?
print("abc".isalpha())    # True
print("abc1".isalpha())   # False

# 알파벳 + 숫자 (AlNum)인지?
print("a1b2".isalnum())   # True
print("a1 b2".isalnum())  # False (공백 때문)

# 대문자/소문자 인지?
print("A".isupper())      # True
print("a".islower())      # True

3. [코딩하기] 비밀번호 가리기

전화번호 뒷자리 4개를 *로 바꾸는 코드를 완성하세요.

phone = "010-1234-5678"
# 5678을 ****로 변경
safe_phone = phone.________("5678", "****")

print(safe_phone) 
# 결과: 010-1234-****
🔑 정답 확인하기 (클릭)

  3. replace: phone.replace("5678", "****")

8️⃣ 꿀팁: 알아두면 쓸모 있는 추가 메서드 🍯

실제 프로그램이나 웹사이트를 만들 때 정말 자주 쓰이는 알짜배기 함수들입니다.

1. 시작과 끝 확인하기 (검사)
  • startswith(): ~로 시작하니? (예: http://)
  • endswith(): ~로 끝나니? (예: .jpg)
url = "http://google.com"
file = "pic.jpg"

print(url.startswith("http")) # True
print(file.endswith(".jpg"))  # True
2. 제목 스타일 만들기 (서식)
  • capitalize(): 문장의 첫 글자만 대문자로
  • title(): 각 단어의 앞글자를 대문자로 (제목처럼)
s = "nice to meet you"

print(s.capitalize()) # "Nice to meet you"
print(s.title())      # "Nice To Meet You"
3. 빈칸 0으로 채우기
  • zfill(길이): 숫자를 문자로 다룰 때(순번 등) 앞자리에 0을 채워줍니다.
print("1".zfill(3))   # "001"
print("12".zfill(3))  # "012"

📌 핵심 요약

  • 길이/개수: len()은 전체 길이, count()는 특정 글자 개수.
  • 찾기: find()는 못 찾으면 -1, index()는 에러!
  • 변환/정리: upper(), lower(), swapcase(), replace(), strip().
  • 검사: isdigit(), isalpha() 등으로 구성 성분 확인.
  • 연결: "구분자".join(리스트)로 리스트를 문자열로 합체!
