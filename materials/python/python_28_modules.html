🧰 파이썬 마스터 코치: 28. 주로 사용하는 모듈

파이썬의 진정한 힘! 거대한 도구함에서 필요한 무기 빌려오기

1️⃣ 모듈(Module)이란? `import`의 마법 🚚

파이썬은 기본적으로 가벼운 상태로 시작합니다. 무거운 도구들은 창고(표준 라이브러리)에 두고, 필요할 때마다 `import`(수입하다/불러오다) 해서 사용합니다.

🛠️ 마스터 코치의 비유: 공구함 빌려오기
• import math: 수학 전용 공구함 전체를 통째로 작업장으로 가져옵니다.
• from math import sqrt: 수학 공구함에서 '루트 계산기(sqrt)' 딱 하나만 꺼내옵니다. 
# 1. 통째로 가져오기 (사용할 때마다 '공구함이름.'을 붙여야 함)
import math
print(math.ceil(3.14))  # math 공구함의 올림(ceil) 도구 사용

# 2. 특정 도구만 쏙 뽑아오기 (이름표 바로 사용 가능)
from math import floor
print(floor(3.14))    # math.floor 라고 안 써도 됨!

2️⃣ `random`: 예측 불가능한 주사위 🎲

게임을 만들거나 테스트 데이터를 뽑을 때 가장 많이 쓰는 모듈입니다. (while문 배울 때 살짝 써봤죠?)

import random

# ① randint(a, b): a 이상 b 이하의 랜덤 정수 하나 뽑기
num = random.randint(1, 10)
print(f"주사위 결과: {num}")  # 예: 7

# ② choice(리스트): 리스트에서 무작위로 '하나만' 골라오기
lunch = ["짜장면", "짬뽕", "볶음밥"]
pick = random.choice(lunch)
print(f"오늘 점심은: {pick}")  # 예: 짬뽕

# ③ shuffle(리스트): 리스트 자체를 뒤섞기 (원본이 바뀜!)
cards = [1, 2, 3, 4, 5]
random.shuffle(cards)
print(f"카드 섞은 결과: {cards}")  # 예: [3, 1, 5, 2, 4]

# ④ sample(리스트, 개수): 리스트에서 '여러 개'를 중복 없이 뽑기
lotto = random.sample(range(1, 46), 6)
print(f"로또 번호: {sorted(lotto)}")  # 예: [5, 12, 23, 31, 38, 42]

# ⑤ uniform(a, b): a 이상 b 이하의 랜덤 실수(소수점) 뽑기
temp = random.uniform(36.0, 37.5)
print(f"체온 측정: {temp:.1f}도")  # 예: 36.8도

# ⑥ random(): 0.0 이상 1.0 미만의 랜덤 실수 (인자 없음!)
chance = random.random()
print(f"확률값: {chance:.4f}")  # 예: 0.7291
📌 정리: random 모듈 핵심 함수 비교
• randint(a, b): 정수 1개 뽑기 (a~b 포함)
• choice(리스트): 리스트에서 1개 뽑기
• shuffle(리스트): 리스트 자체를 섞기 (원본 변경됨!)
• sample(리스트, n): 리스트에서 n개를 중복 없이 뽑기 (원본 유지!)
• uniform(a, b): 실수 1개 뽑기 (a~b)
• random(): 0~1 사이 실수 1개 뽑기

3️⃣ `datetime`: 시간을 다루는 마법사 ⏳

프로그램에서 현재 시간을 기록하거나, 날짜를 계산해야 할 때 꼭 필요한 모듈입니다.

import datetime
from datetime import date, timedelta

# ① datetime.now(): 현재 날짜 + 시간을 한꺼번에 가져오기
now = datetime.datetime.now()
print(now)  # 예: 2026-05-28 10:30:45.123456

# ② strftime(): 날짜/시간을 내가 원하는 형식의 '문자열'로 변환하기
# %Y: 연도(4자리), %m: 월, %d: 일, %H: 시, %M: 분, %S: 초
pretty = now.strftime("%Y년 %m월 %d일 %H시 %M분")
print(pretty)  # 2026년 05월 28일 10시 30분

# ③ strptime(): 반대로 '문자열'을 날짜 객체로 변환하기
text = "2026-12-25"
xmas = datetime.datetime.strptime(text, "%Y-%m-%d")
print(xmas)  # 2026-12-25 00:00:00

# ④ date.today(): 오늘 '날짜만' 깔끔하게 가져오기 (시간 없이)
today = date.today()
print(today)  # 2026-05-28
print(today.year, today.month, today.day)  # 2026 5 28 (각각 꺼내쓰기 가능!)

# ⑤ timedelta(): 날짜 덧셈/뺄셈 계산기 (D-Day 계산!)
# 오늘부터 100일 뒤는 언제일까?
future = today + timedelta(days=100)
print(f"100일 뒤: {future}")  # 예: 2026-09-05

# 크리스마스까지 며칠 남았을까?
xmas_date = date(2026, 12, 25)
remaining = xmas_date - today
print(f"크리스마스까지 {remaining.days}일 남았다!")

# ⑥ weekday(): 요일 알아내기 (0=월, 1=화, ... 6=일)
days_kr = ["월", "화", "수", "목", "금", "토", "일"]
print(f"오늘은 {days_kr[today.weekday()]}요일")
📌 정리: datetime 모듈 핵심 함수 비교
• datetime.now(): 현재 날짜+시간 한꺼번에
• date.today(): 오늘 날짜만 깔끔하게
• strftime(): 날짜 → 문자열 변환 (출력용 꾸미기)
• strptime(): 문자열 → 날짜 변환 (입력값 파싱)
• timedelta(days=n): 날짜 더하기/빼기 계산
• weekday(): 요일 번호 (0=월 ~ 6=일)

4️⃣ `os`와 `sys`: 시스템 통제실 🖥️

파일과 폴더를 다루거나, 파이썬 프로그램 자체를 컨트롤할 때 쓰는 무거운 도구상자입니다.

import os
import sys

# ━━━━━━ os 모듈: 파일/폴더 조종 리모컨 ━━━━━━

# ① getcwd(): 현재 내가 작업 중인 폴더 위치 확인
current_path = os.getcwd()
print(f"내 위치: {current_path}")

# ② listdir(경로): 특정 폴더 안의 파일/폴더 목록 보기
files = os.listdir(".")  # "." = 현재 폴더
print(f"현재 폴더 내용: {files}")

# ③ mkdir(이름): 새 폴더 만들기
# os.mkdir("새폴더")  # 이미 있으면 에러 발생!

# ④ makedirs(경로): 중간 폴더까지 한꺼번에 만들기
# os.makedirs("부모폴더/자식폴더/손자폴더", exist_ok=True)
# exist_ok=True: 이미 있어도 에러 안 남 (안전장치!)

# ⑤ rename(원래이름, 새이름): 파일/폴더 이름 바꾸기
# os.rename("old.txt", "new.txt")

# ⑥ remove(파일명): 파일 삭제 (폴더는 rmdir 사용)
# os.remove("불필요한파일.txt")

# ⑦ os.path: 경로를 안전하게 다루는 하위 도구 모음
print(os.path.exists("test.txt"))    # 파일이 존재하는지? True/False
print(os.path.isfile("test.txt"))    # 파일인지? (폴더면 False)
print(os.path.isdir("폴더이름"))    # 폴더인지?
print(os.path.join("폴더", "파일.txt"))  # 경로를 OS에 맞게 안전하게 합치기
# 결과: 윈도우 → "폴더\파일.txt", 맥/리눅스 → "폴더/파일.txt"

# ━━━━━━ sys 모듈: 파이썬 자체를 조종하는 리모컨 ━━━━━━

# ⑧ sys.exit(): 강제로 프로그램 종료하기
print("이 메시지 다음에는 종료됩니다.")
# sys.exit()  # 이 줄 이후의 코드는 절대 실행 안 됨!

# ⑨ sys.argv: 터미널에서 실행할 때 넘긴 인자(옵션) 목록 보기
# 예: python my_script.py hello world
print(sys.argv)  # ['my_script.py', 'hello', 'world']

# ⑩ sys.version: 현재 파이썬 버전 확인하기
print(sys.version)  # 예: 3.12.0 (main, Oct ...)
📌 정리: os / sys 핵심 함수 비교
• os.getcwd(): 현재 폴더 위치 확인
• os.listdir(경로): 폴더 안 목록 보기
• os.mkdir() / makedirs(): 폴더 생성 (단일/다중)
• os.rename() / remove(): 이름 변경 / 파일 삭제
• os.path.exists() / isfile() / isdir() / join(): 경로 안전 검사 도구
• sys.exit(): 프로그램 강제 종료
• sys.argv: 실행 시 전달된 인자 목록
• sys.version: 파이썬 버전 확인

5️⃣ `math`: 수학 전용 계산기 🧮

기본 사칙연산을 넘어선 고급 수학 계산이 필요할 때 꺼내 쓰는 전문 공구함입니다. import 문법을 설명할 때 잠깐 나왔었죠!

import math

# ① ceil(x): 올림 (천장에 붙이기 → 항상 위로!)
print(math.ceil(3.14))   # 4 (소수점을 무조건 올림)
print(math.ceil(3.01))   # 4

# ② floor(x): 내림 (바닥에 눌러붙이기 → 항상 아래로!)
print(math.floor(3.99))  # 3 (소수점을 무조건 내림)

# ③ sqrt(x): 제곱근(루트) 계산 (√x)
print(math.sqrt(16))    # 4.0 (√16 = 4)
print(math.sqrt(2))     # 1.4142135... (√2)

# ④ pow(x, y): 거듭제곱 (x의 y승)
print(math.pow(2, 10))   # 1024.0 (2의 10승)

# ⑤ fabs(x): 절대값 (항상 양수로 만들기, 실수형 반환)
print(math.fabs(-7.5))  # 7.5

# ⑥ factorial(n): 팩토리얼 (n! = n × (n-1) × ... × 1)
print(math.factorial(5))  # 120 (5! = 5×4×3×2×1)

# ⑦ gcd(a, b): 최대공약수 (Greatest Common Divisor)
print(math.gcd(12, 18))  # 6 (12와 18의 최대공약수)

# ⑧ pi: 원주율 상수 (π = 3.141592...)
print(math.pi)           # 3.141592653589793
radius = 5
area = math.pi * radius ** 2
print(f"반지름 5인 원의 넓이: {area:.2f}")  # 78.54

# ⑨ log(x, 밑): 로그 계산 (밑 생략 시 자연로그 ln)
print(math.log(100, 10))  # 2.0 (log₁₀(100) = 2)
print(math.log2(8))      # 3.0 (log₂(8) = 3, 이진수에서 자주 사용!)
📌 정리: math 모듈 핵심 함수 비교
• ceil(x) / floor(x): 올림 / 내림
• sqrt(x): 제곱근 (루트 √)
• pow(x, y): 거듭제곱 (x의 y승)
• fabs(x): 절대값 (실수형)
• factorial(n): 팩토리얼 (n!)
• gcd(a, b): 최대공약수
• pi: 원주율 상수 3.14...
• log(x, 밑) / log2(x): 로그 계산

📌 핵심 요약

  • 모듈(Module): 복잡하고 유용한 도구들을 담아둔 공구함 세트!
  • import: 공구함을 내 작업공간으로 들고 오라는 주문.
  • from ~ import ~: 전체가 아니라 특정한 도구 장비 딱 하나만 가져와서 바로 쓰는 마법.
  • 자주 쓰는 공구함: 수학은 math, 무작위는 random, 시간은 datetime, 파일/시스템은 os / sys!
