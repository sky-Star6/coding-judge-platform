[모듈 1: 기초 탄탄] 반복문 2단계 (while, break, continue)

1. 💡 눈높이 비유 (The Analogy)

💡 게임의 "살아있는 동안 계속 반복!" 규칙

앞서 배운 for 문이 '딱 10번만 뛰어!' 하고 횟수가 정해져 있을 때 쓴다면, while 문은 '언제 끝날지 정확히는 모르지만 이 조건이 유지되는 동안은 무조건 계속해!' 라고 할 때 사용합니다.

  • **while (조건):** "내 체력이 0보다 큰 동안에는(조건) 계속해서 몬스터를 때려라!(반복)"
  • **break (비상탈출):** "그러다 만약 전설의 보물을 발견하면? 하던 거 다 멈추고 즉시 탈출!"
  • **continue (스킵):** "돌을 주웠네? 이건 쓸모없으니 이번 턴만 패스하고 바로 다음 몬스터 찾아라!"

2. 🔑 핵심 키워드 정리 (Key Terms)

🔑 필수 키워드 3가지
  • ① while (조건식)
    👉 "괄호 안의 조건이 참(true)인 동안 중괄호 안의 코드를 영원히 반복하는 햄스터 쳇바퀴입니다. 주로 while(true)로 무한루프를 만들 때 자주 써요."
  • ② break
    👉 "자판기(switch)에서 썼던 그 브레이크 맞습니다! 반복문 안에서 가장 가까운 쳇바퀴를 강제로 부수고 탈출하는 강력한 기능입니다."
  • ③ continue
    👉 "반복문을 완전히 탈출하진 않고, 밑에 남은 코드를 무시한 채 '다음 바퀴'로 점프(스킵)하게 만듭니다."

3. 💻 실습 코드 & 상세 분석 (Code Snippet)

1부터 차례대로 출력하다가, 특정한 조건에서 스킵(continue)하거나 멈추는(break) 코드를 보겠습니다.

public class LoopWhile {
    public static void main(String[] args) {
        
        int count = 1; // 카운트 변수 준비
        
        // 카운트가 10 이하인 '동안' 영원히 반복!
        while (count <= 10) {
            
            // 만약 카운트가 3이라면?
            if (count == 3) {
                count++; // 무한루프를 막기 위해 숫자를 하나 올려줍니다! (중요)
                System.out.println("숫자 3은 재수 없으니 패스!(continue)");
                continue; // 아래 코드를 안 하고 다음 바퀴로 점프!
            }
            
            // 만약 카운트가 5라면?
            if (count == 5) {
                System.out.println("숫자 5가 나왔다! 강제 탈출!(break)");
                break; // while문을 완전히 박살내고 탈출합니다.
            }
            
            // 정상 출력 파트
            System.out.println("현재 숫자: " + count);
            count++; // while문은 자동으로 숫자를 안 올려주니 우리가 직접 올려야 해요!
            
        } // -- while 닫기 --
        
        System.out.println("프로그램이 무사히 종료되었습니다.");
    }
}
[주의!] ⚠️ 입문자 단골 실수 탑(TOP) 3
  1. 🚨 위험한 무한루프 (Infinite Loop): for문과 달리 while문은 괄호 안에 증감식(i++)이 없습니다! 쳇바퀴 안에서 숫자를 올려주지 않으면, count가 영원히 1에 머물러 컴퓨터가 멈출 때까지 글씨를 쏟아냅니다. 직접 증감식을 넣는 걸 절대 잊지 마세요!
  2. 🚨 continue 직전의 함정: if(count==3) 안에서 continue를 타고 바로 위로 올라가 버리면, 아래에 있던 count++를 만나지 못해 영원히 3인 상태로 무한루프에 빠집니다! 그래서 예제 코드에 continue 직전에 count++를 살짝 넣어둔 것입니다.
  3. 🚨 break 위치: break 밑에는 코드를 쓸 수 없습니다 (Unreachable code). 탈출해버렸으니 컴퓨터가 도달할 수 없거든요!

4. 📝 스스로 챌린지 (Mini Challenge)

진짜 무한루프 while(true)를 이용해 배터리 감소 시뮬레이션을 만들어 봅시다.

📝 미션: 배터리 방전 시뮬레이션

현재 배터리(battery)가 100 있습니다.
while(true) { ... } 라는 영원히 멈추지 않는 무한루프 안에서 한 바퀴 돌 때마다 배터리를 10씩 깎으세요.
그러다 배터리가 0 이하가 되면 "방전되었습니다!"를 출력하고 break로 강제 탈출하는 코드를 작성하세요.

👉 힌트 & 정답(클릭해서 확인하세요)
정답 코드: 
int battery = 100;

while (true) { // 무한 반복!
    System.out.println("스마트폰 사용중... 현재 배터리: " + battery);
    
    battery = battery - 10; // 배터리 10 감소
    
    if (battery <= 0) { // 0과 같거나 작아지면?
        System.out.println("방전되었습니다!");
        break; // 무한루프 강제 탈출!
    }
}
