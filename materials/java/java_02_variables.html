[모듈 1: 기초 탄탄] 변수(Variable)와 서랍장 (기본 자료형)

1. 💡 눈높이 비유 (The Analogy)

💡 변수는 물건을 담는 **이름표 붙은 서랍장**입니다!

우리가 일상생활에서 물건을 정리할 때 어떻게 하나요?

  • 양말을 넣기 위한 칸에는 '양말 서랍'이라고 이름표(변수명)를 붙입니다.
  • 서랍의 모양도 중요합니다. 덩치가 큰 겨울 패딩을 작은 양말 서랍에 억지로 구겨 넣을 순 없으니까요. 컴퓨터 메모리도 마찬가지여서, 데이터의 종류와 크기에 맞는 서랍의 종류(자료형)를 골라야 합니다.
  • 그리고 이 서랍의 내용물은 언제든지 마음대로 빼고 다른 것으로 바꿀 수 있습니다(변하는 수, 변수).

자바에게 "나 정수(int)를 담을 거니까 '나이'라는 이름표를 붙인 서랍장 하나 만들어줘!"라고 명령하는 것이 바로 '변수 선언'입니다.

2. 🔑 핵심 키워드 정리 (Key Terms)

다양한 종류의 서랍장 중에서 자바에서 가장 많이 쓰이는 3대장을 소개합니다!

🔑 필수 키워드 3가지
  • ① int (정수형 서랍장)
    👉 "소수점이 없는 딱 떨어지는 숫자(예: 10, -5, 0)를 담기 위한 기본 서랍장이에요. (나이, 가격, 횟수 등에 사용)"
  • ② double (실수형 서랍장)
    👉 "소수점이 붙어있는 디테일한 숫자(예: 3.14, -0.5)를 담기 위한 넉넉한 서랍장이에요. (키, 몸무게, 원주율 등에 사용)"
  • ③ boolean (논리형 서랍장)
    👉 "오직 '참(true)' 아니면 '거짓(false)' 두 가지 대답만 들어갈 수 있는 아주 작은 서랍장이에요. (스위치의 ON/OFF 상태, 합격/불합격 여부 등에 사용)"

3. 💻 실습 코드 & 상세 분석 (Code Snippet)

직접 서랍장을 만들고 값을 집어넣은 뒤, 다시 수정하여 출력해봅시다!

public class VariableStorage {
    public static void main(String[] args) {
        
        // 1️⃣ 서랍장 만들기 (변수 선언과 동시에 데이터 넣기)
        int age = 20;            // 'age'라는 이름의 정수 서랍에 20을 넣습니다.
        double height = 175.5;   // 'height'라는 이름의 실수 서랍에 175.5를 넣습니다.
        boolean isAdult = true;  // 'isAdult'라는 논리 서랍에 true(참)를 넣습니다.
        
        // 화면에 현재 상태 출력하기
        System.out.println("현재 나이: " + age);
        System.out.println("성인입니까? " + isAdult);
        
        // 2️⃣ 내용물 변경하기 (변수: 변하는 수!)
        // '당해'가 지나서 나이를 한 살 더 먹었다고 가정해볼까요?
        // ⚠️ 주의: 한 번 만든 서랍의 종류(int)는 다시 안 적어도 됩니다! 내용물만 갈아끼우세요.
        age = 21; 
        
        // 바뀐 서랍 속 내용 확인하기
        System.out.println("\n1년 뒤 나이: " + age);
        
    }
}
[주의!] ⚠️ 입문자 단골 실수 탑(TOP) 3
  1. 🚨 서랍 종류 불일치 (Type Mismatch): 정수형 서랍(int)에 소수점(3.14)을 구겨 넣으려고 하거나, 숫자 서랍에 "문자"를 넣으려고 하면 서랍이 부서집니다(에러 발생)! 서랍의 모양과 물건의 모양이 같아야 해요.
  2. 🚨 이름표 중복 부착: 이미 int age = 20; 으로 서랍장을 만들었는데, 뒤에서 또 int age = 21; 이라고 선언하면 컴퓨터가 "어라? 나이라는 이름의 서랍장이 두 개나 있네?" 하고 헷갈려합니다. 값을 바꿀 땐 종류(int)를 빼고 이름표(age)만 부르세요!
  3. 🚨 빈 서랍장 열어보기 (초기화 누락): int level; 이라고 서랍장만 만들고, 아무것도 넣지 않은 상태에서 println(level); 로 열어보려 하면 에러(Not initialized)가 발생합니다. 서랍을 열 땐 항상 내용물이 있어야 해요!

4. 📝 스스로 챌린지 (Mini Challenge)

배운 내용들을 바탕으로 나의 게임 캐릭터 스테이터스 창을 만들어보세요!

📝 미션: 용사의 스탯(상태창) 만들기

아래 3가지 정보를 변수에 담고, 화면에 보기 좋게 출력해 보세요.

  • 1. 용사의 레벨 (예: 10) ➔ 어떤 변수 타입(자료형)이 적절할까요?
  • 2. 현재 이동 속도 (예: 1.5배) ➔ 소수점이 필요합니다!
  • 3. 파티(팀) 가입 여부 (예: 가입안함 = 거짓) ➔ 참/거짓 중 하나겠죠?

(힌트: 문자와 변수를 이어붙일 때는 더하기(+) 기호를 사용하세요!)

👉 힌트 & 정답(클릭해서 확인하세요)
정답 코드 예시: 
// 1. 변수 선언 (서랍 만들기)
int level = 10;
double speed = 1.5;
boolean inParty = false;

// 2. 출력하기 (변수 열어보기)
System.out.println("=== 용사 스테이터스 ===");
System.out.println("LV : " + level);
System.out.println("속도: " + speed);
System.out.println("파티: " + inParty);
💡 해설: println 안에서 "문자" + 변수명을 쓰면, 컴퓨터가 자동으로 서랍 문을 열고 안의 숫자를 문자로 바꿔서 예쁘게 합쳐줍니다!
