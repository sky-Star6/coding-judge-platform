☕ 자바 마스터 교육: 0. 시작하기 (환경 세팅)

우리 함께 즐거운 자바(Java)의 세계로 떠나볼까요? 첫 번째 준비 단계입니다!

1. 💡 눈높이 비유 (The Analogy)

💡 우리는 지금부터 '이탈리안 레스토랑'을 열 거예요!

자바(Java)를 배우는 것은 마치 식당을 열고 요리를 시작하는 것과 같습니다.

2. 🔑 핵심 키워드 정리 (Key Terms)

이번 단원에서 가장 중요한 3가지 용어를 기억하세요. "이게 왜 필요한가?"에 초점을 맞췄습니다!

🔑 필수 키워드 3가지
  • ① JDK (Java Development Kit)
    👉 "자바로 글을 쓰고(컴파일), 만든 프로그램을 실행하게 해주는 모든 도구의 묶음이에요."
  • ② 환경 변수 (Environment Variable)
    👉 "컴퓨터에게 '앞으로 내가 자바라고 부르면 여기가서 도구 가져와!' 하고 지도(경로)를 새겨주는 설정입니다."
  • ③ VS Code (Visual Studio Code)
    👉 "MS에서 만든 강력하고 가벼운 코드 편집기입니다. 자바뿐만 아니라 수많은 언어의 '스마트 맞춤형 노트장' 역할을 해요."

3. ⬇️ 자바(JDK) 다운로드 및 설치하기

자바 프로그램을 만들고 실행하기 위해 가장 뼈대가 되는 JDK를 설치합니다.

  • LTS 버전 선택: 구글에 'Java JDK Download' 혹은 'Amazon Corretto'를 검색합니다. 메인 화면에 있는 'Latest LTS Release' (일반적으로 Java 17 또는 21) 버튼을 클릭하여 설치 파일을 다운로드합니다. (*LTS는 가장 안정적이고 오랫동안 지원받는 버전입니다.)
  • 자동 설정(체크박스): 설치를 진행하다 보면 "Set JAVA_HOME variable" (또는 경로 자동 추가) 같은 옵션 체크박스가 나올 때가 있습니다. 이걸 체크하면 아래 복잡한 환경 변수 설정을 알아서 해줍니다!
  • 수동 설정: 혹시 체크박스가 없었거나, 실수로 체크하지 않고 그냥 'Next'만 눌러 설치했다면? 걱정 마세요! 아래의 수동 설정 가이드를 따라가면 됩니다.

4. �️ 환경 변수 수동 설정하기 (5단계 정복!)

설치할 때 경로 자동 추가를 놓쳤다면, 윈도우(Windows)에게 직접 자바 주소를 가르쳐주어야 합니다.

📍 1단계: 자바가 설치된 경로 복사하기

먼저 자바가 내 컴퓨터 어디에 깔려 있는지 주소를 알아야 합니다.

  1. 키보드의 Windows 키 + E를 눌러 파일 탐색기를 엽니다.
  2. C드라이브 > Program Files > Java (또는 Eclipse Adoptium 등 본인이 설치한 자바 폴더)로 들어갑니다.
  3. jdk-버전명으로 되어 있는 폴더 안으로 들어갑니다.
  4. 탐색기 상단의 주소창 부분을 마우스로 클릭한 뒤, 파랗게 블록이 잡힌 그 경로(예: C:\Program Files\Java\jdk-17)를 복사(Ctrl + C)합니다.

📍 2단계: 환경 변수 설정 창 열기

  1. 윈도우 화면 왼쪽 아래의 검색창(돋보기 모양)에 "환경 변수"라고 검색합니다.
  2. 검색 결과 중 "시스템 환경 변수 편집"을 클릭합니다.
  3. '시스템 속성' 창이 뜨면, 오른쪽 아래에 있는 [환경 변수(N)...] 버튼을 클릭합니다.

📍 3단계: JAVA_HOME 변수 만들기

컴퓨터에게 '자바의 집(설치 폴더)이 여기야'라고 알려주는 과정입니다.

  1. 창 아래쪽의 '시스템 변수(S)' 영역에서 [새로 만들기(W)...] 버튼을 클릭합니다.
  2. 다음과 같이 입력합니다:
    - 변수 이름(N): JAVA_HOME (반드시 대문자로 입력해 주세요!)
    - 변수 값(V): 1단계에서 복사해 둔 경로를 붙여넣기(Ctrl + V) 합니다.
  3. [확인]을 누릅니다.

📍 4단계: Path 변수에 추가하기

컴퓨터가 자바 실행 파일(bin 폴더 안의 파일들)을 바로 찾을 수 있게 길을 뚫어주는 과정입니다.

  1. 여전히 '시스템 변수(S)' 영역에 있는 목록 중에서 Path라는 이름의 변수를 찾아 더블클릭합니다. (또는 선택 후 편집 클릭)
  2. 오른쪽의 [새로 만들기(N)] 버튼을 누릅니다.
  3. 목록의 맨 아래에 빈칸이 생기면 %JAVA_HOME%\bin 이라고 정확하게 입력하고 엔터를 칩니다.
  4. 열려있는 모든 창의 확인 버튼을 눌러 창을 전부 닫아줍니다.

📍 5단계: 설정이 잘 되었는지 확인하기 (중요⭐)

  1. 윈도우 검색창에 cmd를 검색하여 명령 프롬프트를 실행합니다.
  2. 검은 창이 뜨면 java -version을 입력하고 엔터를 칩니다. 자바 버전 정보가 쭉 나오면 1차 성공입니다!
  3. 추가로 javac -version을 입력하고 엔터를 칩니다. 역시 버전 정보가 나오면 환경 변수 설정이 완벽하게 끝난 것입니다! 🎉

(주의: 만약 VS Code나 명령 프롬프트가 이미 켜져 있었다면, 방금 한 설정이 반영될 수 있도록 해당 프로그램을 껐다가 다시 켜주세요.)

5. 🛠️ VS Code로 자바 코딩하기 (IDE 세팅)

비싼 도구 없이도, 아주 강력한 무료 에디터인 Visual Studio Code를 통해 자바 프로그램을 작성하고 실행해 보겠습니다!

  • 1단계 (설치): 구글에 "VS Code Download"를 검색하여 공식 홈페이지에서 설치합니다.
  • 2단계 (확장팩 추가): VS Code를 열고 왼쪽 메뉴 중 블록 모양 아이콘(Extensions, 단축키 Ctrl+Shift+X)을 클릭합니다.
  • 3단계 (자바 마법사 소환 🪄): 검색창에 "Extension Pack for Java"를 검색합니다. (Microsoft에서 제공하는 빨간색 커피잔 아이콘입니다.)
  • 4단계 (설치 완료): 'Install' 버튼을 눌러 설치하면, VS Code가 자바 전용 도우미로 변신합니다!

4. 💻 실습 코드 & 상세 분석 (Code Snippet)

VS Code에서 Main.java라는 파일을 하나 만들고, 자바의 가장 표준적인 첫 코드를 작성해 볼까요?

public class Main {
    // 자바 프로그램이 시작될 때 가장 먼저 실행되는 대문입니다 (메인 메서드)
    public static void main(String[] args) {
        
        // 'System.out'은 모니터 화면을, 'println'은 화면에 줄을 바꿔서 글을 찍어준다는 뜻이에요.
        // 주의: 안에 적을 문구는 반드시 쌍따옴표(" ")로 감싸야 합니다.
        System.out.println("Hello, World!");

        // 자바에서는 명령이 끝날 때마다 항상 세미콜론(;)을 마침표처럼 찍어야 한답니다!
        System.out.println("반갑습니다, 자바 초보 마스터입니다!");
    }
}
[주의!] ⚠️ 입문자 단골 실수 탑(TOP) 3

이 부분이 핵심이에요! 처음 자바를 접할 때 여기서 에러를 가장 많이 만납니다.

  1. 🚨 대소문자의 헷갈림: System과 String은 반드시 **대문자(S)**로 시작해야 해요! 소문자로 적으면 에러(Error)가 발생합니다.
  2. 🚨 세미콜론 누락: 문장의 끝인 세미콜론(;)을 빼먹어서 빨간 줄이 나타나는 경우가 절반 이상이에요.
  3. 🚨 클래스명과 파일명의 불일치: class Main 이라고 코드를 적었다면, 저장하는 파일 이름도 반드시 Main.java 여야만 합니다!

자신의 실력을 점검해 보세요. 직접 생각하고 코드를 수정하는 과정에서 실력이 쑥쑥 자랍니다!

📝 첫 번째 미션: 자기소개 출력하기

위에서 배운 System.out.println();을 응용해서, 화면에 자신의 이름과 좋아하는 취미가 두 줄에 걸쳐서 나오도록 코드를 추가해 보세요.

내 취미는 코딩하기야.
