[모듈 1: 기초 탄탄] 문자와 문자열 처리 (char vs String)

1. 💡 눈높이 비유 (The Analogy)

💡 알파벳 블록 한 개와 블록으로 만든 기차!

문자와 문자열의 차이는 장난감 블록과 같습니다.

  • **문자(char):** 딱 한 개의 알파벳 블록입니다. 'A', '가', '3' 처럼 무조건 한 글자만 올 수 있고, 홑따옴표(' ')로 감싸서 표시합니다.
  • **문자열(String):** 여러 개의 블록을 실로 꿰어 연결한 블록 기차(단어/문장)입니다. "Hello", "자바 마스터" 처럼 여러 글자를 담을 수 있으며, 쌍따옴표(" ")로 감쌉니다. (글자가 한 개여도, 0개여도 기차로 만들 수 있습니다!)

자바에서는 이 둘을 엄격하게 구분하므로, 홑따옴표와 쌍따옴표의 쓰임새를 정확히 알아야 합니다.

2. 🔑 핵심 키워드 정리 (Key Terms)

🔑 필수 키워드 3가지
  • ① char (문자 타입)
    👉 "오직 단 1개의 글자만 저장하는 작은 서랍. 반드시 홑따옴표(' ')를 사용해야 해요. (예: 성적 'A', 성별 'M')"
  • ② String (문자열 타입)
    👉 "여러 글자가 이어진 문장을 저장하는 큰 창고. 반드시 쌍따옴표(" ")를 사용해야 해요. (예: 이름 "홍길동", 주소 "서울시")"
    🚨 주의: 다른 기본 서랍들과 달리 String은 대문자 'S'로 시작합니다!
  • ③ 문자열 결합 연산자 (+)
    👉 "숫자끼리 더하면 산수 덧셈이 되지만, 문자열에 더하기(+)를 쓰면 앞뒤 글자를 강력한 자석처럼 이어 붙여줍니다!"

3. 💻 실습 코드 & 상세 분석 (Code Snippet)

문자와 문자열을 선언하고, 서로 이어 붙이는(결합) 방법을 코드로 확인해 봅시다.

public class StringExample {
    public static void main(String[] args) {
        
        // 1️⃣ char: 홑따옴표로 단 한 글자만!
        char grade = 'A';
        
        // 2️⃣ String: 쌍따옴표로 문장 만들기 (명심: S는 대문자!)
        String name = "김자바";
        String greeting = "안녕하세요!";
        
        // 3️⃣ 문자열 결합 (+ 연산자 활용)
        // 글자와 글자를 더하면 문장이 이어집니다.
        String message = name + "님, " + greeting;
        System.out.println(message);
        
        // 글자와 숫자를 더하면? 숫자가 자동으로 글자로 변해서 이어붙습니다!
        int score = 100;
        System.out.println(name + "의 자바 점수는 " + score + "점, 등급은 " + grade);
        
    }
}
[주의!] ⚠️ 입문자 단골 실수 탑(TOP) 3
  1. 🚨 따옴표 혼동: char c = "A"; (쌍따옴표 사용) 또는 String s = 'Hello'; (홑따옴표 사용) 처럼 반대로 쓰면 즉시 에러가 발생합니다.
  2. 🚨 char에 두 글자 쑤셔넣기: char c = 'AB'; 는 불가능합니다. 블록 한 칸에 블록 두 개를 넣을 수 없습니다!
  3. 🚨 String 소문자 시작: string name = "홍길동"; 자바에서 string(소문자)이라는 서랍은 없습니다. 반드시 대문자 String을 써야 합니다! (String은 특별한 손님(클래스형 변수)이기 때문에 대문자로 모십니다.)

📝 미션: 문자열 퍼즐 맞추기

아래 변수들을 제공합니다. 이 변수들을 + 연산자로만 조합해서 "내 이름은 스파이더맨, 나이는 20살이며 성별은 M입니다." 라는 문장이 출력되게 만들어 보세요.

  • String heroName = "스파이더맨";
  • int heroAge = 20;
  • char heroGender = 'M';
👉 힌트 & 정답(클릭해서 확인하세요)
정답 코드: 
String heroName = "스파이더맨";
int heroAge = 20;
char heroGender = 'M';

System.out.println("내 이름은 " + heroName + ", 나이는 " + heroAge + "살이며 성별은 " + heroGender + "입니다.");
💡 해설: 띄어쓰기도 훌륭한 '문자'입니다! 변수와 한글을 연결할 때 쌍따옴표 안의 띄어쓰기("살이며 ")를 신경 써주면 문장이 예쁘게 출력됩니다.
