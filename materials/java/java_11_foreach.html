[모듈 2: 데이터 그룹화] 향상된 for문 (for-each) - 배열을 더 짧게 훑기

1. 💡 눈높이 비유 (The Analogy)

💡 컨테이너 벨트 위의 박스 하나씩 꺼내기!

기존의 전통적인 for문은 "0번부터 시작해서, 배열 길이보다 작을 때까지, 1칸씩 이동해!"라며 아주 깐깐하고 귀찮은 조건(i=0; i<len; i++)을 적었습니다.

하지만 향상된 for문(for-each 구문)은 '컨베이어 벨트'입니다.

  • 이 벨트(배열) 위에 짐이 10개가 있든 100개가 있든 신경 끄세요.
  • 그냥 "처음부터 끝까지 알아서 돌면서 박스(데이터)를 무조건 한 개씩 내 손에 쥐여줘!" 라고 명령하는 아주 쿨하고 편리한 방식입니다.

2. 🔑 핵심 키워드 정리 (Key Terms)

🔑 필수 키워드 2가지
  • ① 향상된 for문 문법: for (타입 꺼낼변수명 : 배열)
    👉 "콜론(:)을 기준으로, 오른쪽엔 배열 덩어리를 두고, 왼쪽엔 그 배열 안의 알맹이를 하나씩 받아먹을 빈 그릇(변수)을 준비합니다. (int i=0; ...) 같은 깐깐한 조건식이 완벽히 사라집니다!"
  • ② 순회(Iteration) 전용
    👉 "이 문법은 오직 배열(이나 차후 배울 컬렉션)의 들어있는 모든 물건을 '처음부터 끝까지 빠짐없이' 확인하고 싶을 때만 쓸 수 있는 마법 주문입니다."

3. 💻 실습 코드 & 상세 분석 (Code Snippet)

과일 상자에서 과일을 하나씩 꺼내는 과정을 전통 방식과 향상된 방식으로 비교해 봅시다.

public class EnhancedFor {
    public static void main(String[] args) {
        
        // 과일을 담은 문자열 배열 준비
        String[] fruits = {"사과", "바나나", "포도", "오렌지"};
        
        
        // ❌ 1. 전통적인 for 방식 (번거로움, 실수하기 딱 좋음)
        System.out.println("--- 전통 방식 ---");
        for (int i = 0; i < fruits.length; i++) {
            System.out.println("꺼낸 과일: " + fruits[i]);
        }
        
        
        // ✅ 2. 향상된 for-each 방식 (깔끔함, 세련됨)
        System.out.println("\n--- 향상된 for 방식 ---");
        // 해석: fruits 배열 안에서 값을 무조건 순서대로 하나씩 뽑아 f 라는 변수에 담습니다!
        for (String f : fruits) {
            System.out.println("맛있는 과일: " + f);
        }
        
    }
}
[주의!] ⚠️ 입문자 단골 실수 탑(TOP) 3
  1. 🚨 타입(Type) 불일치: 과일 배열(String[])을 돌리면서 받아먹는 그릇을 int f라고 잘못 만들면 에러가 납니다. 나오는 물건의 형태와 받아줄 서랍의 형태가 완벽히 같아야 해요.
  2. 🚨 순서를 거꾸로 하거나 건너뛰기 불가: 향상된 for문은 오직 0번부터 마지막 번호까지 "전부다 1장씩 직진!"만 가능합니다. "나는 뒤에서부터 뽑고 싶어!" 혹은 "짝수 번호 방만 보고 싶어!" 할 때는 쓸 수 없습니다. 그때는 전통적인 for문을 써야 합니다.
  3. 🚨 콜론(:) 빼먹기 주의: 향상된 for문의 상징인 : (콜론) 대신 for(String f = fruits) 처럼 =를 찍으면 에러가 납니다!

📝 미션: 영수증 총합 계산기

내가 어제 쓴 간식비 배열 int[] prices = {2000, 4500, 3000, 1500}; 가 주어졌습니다.
향상된 for문을 사용해 이 영수증들(배열) 안의 돈을 빈 변수 total에 전부 계속해서 더하여(누적 합산), 최종 돈의 총합을 출력해 보세요!

👉 힌트 & 정답(클릭해서 확인하세요)
정답 코드: 
int[] prices = {2000, 4500, 3000, 1500};
int total = 0; // 합산을 기록할 저금통 준비

// prices 배열에 있는 과자값을 꺼내 p에 하나씩 담으며 무한 반복!
for (int p : prices) {
    total = total + p; // 꺼낸 돈(p)을 저금통(total)에 계속 더하자!
}

System.out.println("어제 쓴 총 간식비: " + total + "원");
