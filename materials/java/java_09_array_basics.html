[모듈 2: 데이터 그룹화] 배열(Array) 기초 - 1차원 캐비닛 만들기

1. 💡 눈높이 비유 (The Analogy)

💡 번호표가 붙은 목욕탕의 길쭉한 캐비닛!

지금까지 배운 변수가 딱 하나의 물건만 넣을 수 있는 '작은 종이 상자'라면, 학생 100명의 점수를 저장하려면 상자 100개가 필요해서 방이 꽉 찰 겁니다.

그래서 우리는 위아래로 철썩 붙어있는 거대한 캐비닛 세트(배열, Array)를 들여놓습니다!

  • 이 거대한 캐비닛에는 "모두 같은 종류의 물건(타입)"만 넣을 수 있습니다. 신발장엔 신발만, 옷장엔 옷만!
  • 캐비닛의 각 칸에는 "몇 번째 칸인지 알려주는 번호표(인덱스)"가 붙어 있어서 쉽게 찾을 수 있습니다.
  • ⚠️ 중요한 사실: 컴퓨터는 1부터가 아니라 0부터 숫자를 세기 시작합니다! (첫 번째 칸 = 0번 칸)

2. 🔑 핵심 키워드 정리 (Key Terms)

🔑 필수 키워드 3가지
  • ① 배열 선언 ( int[] )
    👉 "자료형 뒤에 대괄호 [] 를 붙이면 '이건 단칸방이 아니라 여러 칸이 붙은 아파트(배열)다!'라고 선언하는 표식입니다."
  • ② new 연산자
    👉 "자바야! 메모리 땅에다 나를 위해 new(새로운) 공간 좀 크게 하나 지어줘! 라고 부탁하는 주문입니다. 배열의 방 개수를 정할 때 씁니다."
  • ③ 인덱스 (Index)
    👉 "배열의 칸마다 붙어있는 방 번호표입니다. 무조건 0번부터 시작한다는 규칙을 외우지 않으면 평생 에러에 시달립니다!"

3. 💻 실습 코드 & 상세 분석 (Code Snippet)

5명의 학생 점수를 담는 배열(캐비닛)을 만들고 값을 넣어봅시다.

public class ArrayBasic {
    public static void main(String[] args) {
        
        // 방법 1: 빈 방 5개짜리 배열 아파트 먼저 짓기
        int[] scores = new int[5]; // 정수를 담을 빈 방 5칸을 세팅!
        
        // 각 방마다 번호표(0부터 시작)를 부르고 값을 넣습니다.
        scores[0] = 85;  // 첫 번째 칸 (0번 방)
        scores[1] = 90;  // 두 번째 칸
        scores[2] = 100; // 세 번째 칸
        scores[3] = 75;  // 네 번째 칸
        scores[4] = 88;  // 다섯 번째 칸 (길이가 5면, 마지막 방 번호는 무조건 4입니다!)
        
        // 방법 2: 배열을 지음과 동시에 값 채워넣기 (초보자 추천★)
        // 중괄호 { } 안에 값을 쉼표로 나열하면 알아서 방 개수가 맞춰집니다.
        String[] friends = {"철수", "영희", "민수"};
        
        // 꺼내서 써보기
        System.out.println("나의 첫 번째 베프는? " + friends[0]); // 철수 출력
        
        // 배열과 for문은 영혼의 단짝입니다!
        System.out.println("\n[학생 점수 출력]");
        for (int i = 0; i < 5; i++) {
            System.out.println(i + "번 방의 점수: " + scores[i]);
        }
        
    }
}
[주의!] ⚠️ 입문자 단골 실수 탑(TOP) 3
  1. 🚨 방 번호 초과 (ArrayIndexOutOfBoundsException): 자바에서 가장 많이 보게 될 에러 중 하나입니다! 방을 5개 만들었으므로 마지막 방 번호는 4입니다. scores[5]를 열어달라고 하면 "그런 방은 없다!"며 즉시 컴퓨터가 뻗어버립니다.
  2. 🚨 배열 이름만 덩그러니 출력: System.out.println(friends); 이렇게 캐비닛 자체를 출력해버리면 안의 값이 나오는 게 아니라, 캐비닛의 '이상한 메모리 주소값(@7a81197d)'이 출력됩니다. 안의 내용물이 보고 싶다면 반드시 대괄호 [번호]를 써서 하나의 방을 콕 집어줘야 합니다.
  3. 🚨 방 크기 변경 불가: new int[5] 로 아파트를 한 번 지어버리면, 나중에 "학생이 1명 더 왔으니 방 하나 더 늘려줘!" 가 불가능합니다. 배열의 크기는 태어날 때 정해지면 절대 안 바뀝니다. (나중에 유연한 'List'를 배우면 해결됩니다!)

4. 📝 스스로 챌린지 (Mini Challenge)

선생님이 되어 학생들의 영어 단어 시험 점수 평균을 내볼까요?

📝 미션: 배열의 점수 합산하기

학생 3명의 영어 점수가 각각 80, 90, 100점입니다.
1) 이 값을 담는 정수형(int) 배열 변수 engScores를 중괄호 { }를 써서 한 줄로 만드세요.
2) 세 방에 든 점수들을 각각 꺼내어([] 사용) 더한 뒤 총합과 평균을 출력해 보세요.

👉 힌트 & 정답(클릭해서 확인하세요)
정답 코드: 
// 한 줄로 선언과 동시에 데이터 채우기
int[] engScores = {80, 90, 100};

// 0번방, 1번방, 2번방 꺼내서 더하기
int total = engScores[0] + engScores[1] + engScores[2];
int avg = total / 3; // 사람 수가 3명이니까 3으로 나눔

System.out.println("영어 점수 총합: " + total);
System.out.println("영어 점수 평균: " + avg);
