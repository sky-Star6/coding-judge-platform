[모듈 1: 기초 탄탄] 반복문 1단계 (for) - 정해진 횟수만큼 반복하기

1. 💡 눈높이 비유 (The Analogy)

💡 체육 선생님의 운동장 달리기 지시!

컴퓨터가 가장 잘하는 일은 '똑같은 일을 지치지 않고 수만 번 반복하는 것'입니다. for문은 체육 선생님이 학생들에게 내리는 명확한 지시와 같습니다.

  • **시작점:** "1번 학생부터 시작해!"
  • **도착점(조건):** "10번 학생이 될 때까지만 뛰어!"
  • **보폭(증감):** "한 번 뛸 때마다 번호를 1씩 올려가면서 뛰어!"

즉, "언제부터(초기화) ~ 언제까지(조건) ~ 얼만큼씩(증감)" 반복할지가 명확하게 정해져 있을 때 사용하는 최고의 도구입니다.

2. 🔑 핵심 키워드 정리 (Key Terms)

🔑 필수 키워드 3가지
  • ① 초기화 (Initialization)
    👉 "루프(반복)를 돌리기 위한 카운트다운 타이머(변수)를 세팅합니다. 주로 int i = 0 또는 int i = 1 로 시작합니다. (i는 index의 약자)"
  • ② 조건식 (Condition)
    👉 "타이머가 어디까지 가야 멈출 것인지 정합니다. 예: i < 10 (i가 10보다 작을 때까지만 뺑뺑이를 돌아라!)"
  • ③ 증감식 (Increment/Decrement)
    👉 "한 바퀴 돌 때마다 타이머를 1씩 올릴지(i++) 내릴지(i--) 정합니다. 이거 안 쓰면 무한루프(영원히 안 끝남)에 빠져요!"

3. 💻 실습 코드 & 상세 분석 (Code Snippet)

1부터 5까지 순서대로 세어주는 프로그램을 `for`문을 사용해 단 3줄로 작성해 봅시다.

public class LoopFor {
    public static void main(String[] args) {
        
        // for ( 1.초기세팅 ; 2.종료조건 ; 3.매바퀴증감 )
        for (int i = 1; i <= 5; i++) {
            
            // 이 괄호 안의 코드가 5번 반복 실행됩니다.
            System.out.println(i + "번째 달리기 중입니다!");
            
        } // 여기서 다시 위로 올라가서 i를 1 증가(i++)시키고 조건을 검사합니다.
        
        System.out.println("운동 끝!");
        
    }
}
[주의!] ⚠️ 입문자 단골 실수 탑(TOP) 3
  1. 🚨 콤마(,)가 아니라 세미콜론(;): 괄호 안의 3가지 요소 사이에는 반드시 마침표 역할을 하는 세미콜론(;)이 들어가야 합니다. 쉼표(,)를 쓰면 에러가 납니다! for (int i=1, i<5, i++) // ❌틀림!
  2. 🚨 증감식에 세미콜론 찍기: 마지막 증감식 뒤에는 세미콜론을 찍지 않습니다. for (int i=1; i<=5; i++;) // ❌틀림!
  3. 🚨 i의 범위를 착각해요: i < 5 는 1, 2, 3, 4 까지만 실행됩니다. 숫자 5까지 꼭 포함하고 싶다면 작거나 같다인 <= 5를 쓰거나, < 6 으로 써야 한다는 점을 명심하세요!

4. 📝 스스로 챌린지 (Mini Challenge)

구구단 2단을 출력해 주는 반복문을 만들어 볼까요?

📝 미션: 구구단 2단 자판기

for문을 1번부터 9번까지(1 ~ 9) 반복하도록 설정하고, 2 * i의 결과를 화면에 출력해 보세요.

출력 화면 예시:
2 * 1 = 2
2 * 2 = 4
...
2 * 9 = 18

👉 힌트 & 정답(클릭해서 확인하세요)
정답 코드: 
int dan = 2;

// 숫자 i가 1부터 9까지 1씩 증가하며 9번 돕니다.
for (int i = 1; i <= 9; i++) {
    // 문자열 결합을 이용해 예쁘게 찍어줍니다.
    System.out.println(dan + " * " + i + " = " + (dan * i));
}
