Module 1. 판다스(Pandas) 분석 기초

Module 2. 전처리와 핵심 그래프 시각화

🔍 1단계: 안전하게 데이터 불러오기 (EDA)

실제 엑셀 데이터를 불러오고 제대로 들어왔는지 요약 정보를 탐색(EDA)합니다.

💡 엑셀 파일 넣는 방법: 실습에 사용할 sales_data.xlsx 파일을 여러분이 파이썬 코드를 작성하는 파일(예: main.py)과 반드시 같은 폴더(디렉토리) 안에 복사해서 넣어주세요! 그래야 파이썬이 파일을 바로 찾을 수 있습니다.
⚠️ 주의사항: 파일을 열 때 파일이 없으면 에러가 나서 프로그램이 다운됩니다. os.path.exists()를 사용해 파일이 있는지 사전에 검사하는 것도 좋은 실무 습관입니다.
PYTHON 
import pandas as pd
import os

file_name = 'sales_data.xlsx'

if os.path.exists(file_name):
    # 엑셀 파일 읽기
    df = pd.read_excel(file_name)
    print(f"✅ 로드 성공! 크기: {df.shape}")
    
    # 탐색적 데이터 분석 (EDA) 기초
    print(df.shape)       # 표의 크기 확인 (행, 열 개수)
    print(df.info())      # 데이터 타입, 비어있는 값 유무 확인
    print(df.describe())  # 기초 통계 요약
    print(df.head())      # 처음 5줄 미리보기
else:
    print("❌ 에러: 파일을 찾을 수 없습니다.")

🎯 2단계: 데이터 선택 (Data Selection)

수많은 데이터 중에서 분석에 진짜 필요한 열(Column)만 쏙쏙 골라냅니다.

PYTHON 
# 1. 원하는 열(Column)만 선택하기
subset_df = df[['품목', '수량', '단가', '매출액']]

# 2. 원하는 행(Row)만 선택하기 (예: 처음부터 5번째 줄까지)
# iloc는 인덱스 번호를 기반으로 데이터를 선택하는 강력한 기능입니다.
row_subset = df.iloc[0:5]

🔍 3단계: 데이터 필터링 (Data Filtering)

내가 원하는 조건에 딱 맞는 데이터(행, Row)만 남기는 기술입니다. 엑셀의 '필터' 기능과 완벽하게 동일합니다.

PYTHON 
# 1. AND 조건(&): 수량이 5개 이상 '이고' 지역이 '서울'인 경우
and_df = df[(df['수량'] >= 5) & (df['지역'] == '서울')]

# 2. OR 조건(|): 지역이 '서울' '이거나' '부산'인 경우 (역슬래시 키)
or_df = df[(df['지역'] == '서울') | (df['지역'] == '부산')]

# 3. isin() 활용: OR 조건이 많을 때 훨씬 깔끔하게 쓰는 방법!
isin_df = df[df['지역'].isin(['서울', '부산', '경기'])]
print(isin_df)

🧹 4단계: 데이터 전처리 (Data Cleansing)

현업의 데이터는 깨끗하지 않습니다. 빈 값을 0으로 채워주거나 쓸모없는 데이터를 지우는 정제 과정입니다.

PYTHON 
# 1. 빈 칸(결측치, NaN)을 숫자 0으로 안전하게 채우기
df = df.fillna(0)

# 2. 비어있는 값이 하나라도 있는 행을 아예 삭제하기
# df = df.dropna()

# 3. 완벽하게 똑같은 중복 데이터(행) 지우기
df = df.drop_duplicates()

⚙️ 5단계: 데이터 변환 및 연산

기존에 있는 열의 데이터를 활용해 계산을 수행하고 새로운 열을 만들어냅니다.

PYTHON 
# 1. 수량과 단가를 곱해 '계산된_매출액' 열 만들기
df['계산된_매출액'] = df['수량'] * df['단가']

# 2. 문자열 '서울'을 'Seoul'로 바꾸기 (replace)
df['지역'] = df['지역'].replace('서울', 'Seoul')

# 3. 데이터 타입 변경 (예: 실수를 정수로 변경) - 실무에서 정말 많이 씁니다!
df['수량'] = df['수량'].astype(int)

📊 6단계: 데이터 요약 (Data Summary)

전체 데이터의 총합, 평균 등 요약된 기초 통계 결과를 단 한 줄로 뽑아냅니다.

PYTHON 
# 합계(sum), 평균(mean), 개수(count), 최댓값(max), 최솟값(min)
print(f"총 수량: {df['수량'].sum()}개")
print(f"평균 단가: {df['단가'].mean()}원")
print(f"최대 매출액: {df['매출액'].max()}원")
print(f"총 거래 건수: {df['주문번호'].count()}건")

↕️ 7단계: 데이터 정렬 (Data Sorting)

데이터를 오름차순이나 내림차순으로 예쁘게 줄 세우는 기능입니다. 순위표를 만들 때 필수적이죠.

PYTHON 
# '매출액'을 기준으로 내림차순(큰 금액이 위로) 정렬하기
# ascending=False 옵션을 주면 내림차순이 됩니다. (기본값은 True, 오름차순)
sorted_df = df.sort_values(by='매출액', ascending=False)
print(sorted_df.head())

📦 8단계: 데이터 그룹화 (Data Grouping)

엑셀의 '피벗 테이블(Pivot Table)' 기능과 같습니다. 특징별로 묶어 분석할 때 최고입니다.

PYTHON 
# '지역' 별로 데이터를 그룹화하여 '매출액'의 합계를 구하기
region_sales = df.groupby('지역')['매출액'].sum()
print(region_sales)

# 결과를 깔끔한 데이터프레임 형식으로 다시 만들기 (reset_index)
region_sales_df = region_sales.reset_index()
print(region_sales_df)

지금까지 배운 8단계를 모두 결합하여 실제 현업에서 쓰이는 전체 코드 흐름 3가지를 보여드립니다. 그대로 복사해서 실행해 보세요!

🔥 예제 1: 특정 품목의 고액 결제 건만 뽑아서 정렬하기
PYTHON 
import pandas as pd

# 1. 파일 로드 및 전처리
df = pd.read_excel('sales_data.xlsx')
df = df.dropna() # 결측치 제거

# 2. 필터링: '노트북' 이면서 매출액이 '100만원' 이상인 데이터
target_df = df[(df['품목'] == '노트북') & (df['매출액'] >= 1000000)]

# 3. 정렬 및 선택: 매출액 기준 내림차순 정렬 후 상위 5개만 보기
result = target_df.sort_values(by='매출액', ascending=False).head(5)

🔥 예제 2: 지역별 총 매출액과 평균 단가 요약 보고서
PYTHON 
import pandas as pd

df = pd.read_excel('sales_data.xlsx')
df = df.fillna(0)

# 그룹화(groupby) 시 여러 개의 열(매출액, 단가)에 각각 다른 연산(sum, mean) 적용하기
# agg() 함수는 현업 필수 스킬입니다!
summary_df = df.groupby('지역').agg({
    '매출액': 'sum',   # 지역별 총 매출액
    '단가': 'mean'    # 지역별 평균 단가
}).reset_index()

# 보기 좋게 내림차순 정렬
summary_df = summary_df.sort_values(by='매출액', ascending=False)
print(summary_df)
🔥 예제 3: 단가 인상 시뮬레이션 및 데이터 내보내기
PYTHON 
import pandas as pd

df = pd.read_excel('sales_data.xlsx')

# 1. 데이터 변환: 내년도 물가 상승을 반영하여 모든 품목 단가 10% 인상
df['예상_단가'] = df['단가'] * 1.1

# 2. 연산: 예상 매출액 재계산 (수량 * 예상단가)
df['예상_매출액'] = df['수량'] * df['예상_단가']

# 3. 데이터 타입 깔끔하게 정수로 통일 (소수점 제거)
df['예상_단가'] = df['예상_단가'].astype(int)
df['예상_매출액'] = df['예상_매출액'].astype(int)

# 4. 필수 컬럼만 선택하여 새로운 엑셀 파일로 저장
final_df = df[['품목', '수량', '예상_단가', '예상_매출액']]
final_df.to_excel('sales_forecast_2025.xlsx', index=False)
print("🎉 내년도 예상 매출액 엑셀 파일이 자동 생성되었습니다!")

Module 3. 데이터 시각화 핵심 (Matplotlib 기초)

💡 라이브러리 임포트: 그래프를 그리기 위해 항상 import matplotlib.pyplot as plt를 선언하고 시작합니다.

📈 1단계: 선 그래프 기초 (Plot & X, Y 좌표)

데이터의 추세를 선으로 이어 그리는 가장 기본적인 그래프입니다.

▲ 기본적으로 리스트 하나만 넣으면 Y값으로 자동 인식하여 대각선 형태의 그래프가 그려집니다.

PYTHON 
import matplotlib.pyplot as plt

# 1. 리스트가 하나면 Y값으로 자동 인식 (X는 0, 1, 2, 3... 자동 생성)
plt.plot([1, 2, 3, 4])
plt.show()

# 2. X와 Y 좌표 지정하기 (두 개의 리스트를 입력하면 순서대로 X, Y로 인식)
# plt.plot([X값 리스트], [Y값 리스트])
plt.plot([1, 2, 3, 4], [1, 4, 9, 16])
plt.show()

🏷️ 2단계: 그래프 의미 부여 및 뷰포트 조정

그래프에 제목과 X/Y축 이름을 달고, 보여줄 축의 범위(Range)를 설정합니다.

▲ X축과 Y축에 이름이 달리고, axis() 함수 덕분에 설정한 범위(X: 0~5, Y: 0~20)까지만 렌더링된 모습입니다.

PYTHON 
plt.plot([1, 2, 3, 4], [1, 4, 9, 16])

# 1. 레이블 설정
plt.xlabel('x-label')
plt.ylabel('y-label')
plt.title('Graph Title')

# 2. 축 범위 지정 (Range) - 반드시 4개의 값을 리스트로 입력
# plt.axis([xmin, xmax, ymin, ymax])
plt.axis([0, 5, 0, 20])

plt.show()

✨ 3단계: 포인트 강조와 색채 더하기

▲ 동일한 그래프에 마커를 달거나 색상 이름(springgreen), 혹은 정밀한 Hex 코드 색상을 넣을 수 있습니다.

PYTHON

🧮 4단계: Numpy를 활용한 자동 데이터 생성

일정한 간격으로 수많은 데이터를 생성하여 자연스러운 곡선을 그릴 때 필수적인 Numpy의 활용법입니다.

▲ Numpy로 만든 미세한 간격의 데이터 포인트를 이어 그려 아주 매끄러운 이차방정식 곡선이 그려졌습니다.

PYTHON 
import numpy as np

# 0부터 2 미만까지 0.2 간격으로 배열 생성 (0.0, 0.2, 0.4 ... 1.8)

# 곡선 그래프 (y값은 x의 제곱)
plt.plot(x, x**2)
plt.show()

📊 5단계: 다양한 목적의 그래프들 (막대, 산점도, 파이)

목적에 맞는 여러 시각화 방식을 적용해 봅니다.

1. 수량 비교: 막대 그래프 (Bar Graph)

▲ plt.bar()는 세로로, plt.barh()는 가로로 막대를 그려 수량을 직관적으로 비교할 수 있습니다. 

years = ['2017', '2018', '2019']
values = [100, 400, 900]

# 세로 막대 (plt.bar)
plt.bar(years, values, width=0.5)

# 가로 막대 (plt.barh)
plt.barh(years, values, height=0.5)
2. 상관관계와 분포: 산점도 (Scatter Plot)

▲ plt.scatter()를 사용하여 점의 크기(s)와 투명도(alpha)를 조절해 데이터 분포를 파악할 수 있습니다. 

x = np.random.rand(30)
y = np.random.rand(30)

# s=면적(Size), c=색상(Color), alpha=투명도(0~1)
plt.scatter(x, y, s=100, c='hotpink', alpha=0.5)
3. 비율 시각화: 파이 차트 (Pie Chart)

▲ explode를 설정해 특정 항목(Math)만 조각을 떼어내 강조하는 효과를 주었습니다. 

ratio = [34, 32, 16, 18]
labels = ['Info', 'Math', 'Sci', 'Phys']

# 특정 조각 돌출시키기 (Math 항목만 0.1만큼 돌출)
explode = (0, 0.1, 0, 0)

# autopct는 파이 안에 퍼센트를 자동으로 표시하는 옵션
plt.pie(ratio, labels=labels, explode=explode, autopct='%.1f%%')

🧑‍💻 종합 실습: 다중 그래프 그리기

지금까지 배운 옵션을 모두 조합하여 하나의 화면에 여러 그래프와 범례(Legend)를 추가합니다.

▲ 각 데이터 선마다 고유한 마커(별, 오각형, 사각형)와 라벨을 부여하고, 우측 상단에 범례를 표시했습니다. 

x = np.arange(0, 25, 1.5)
y1 = np.random.randn(17) * 5
y2 = np.random.randn(17) * 5
y3 = np.random.randn(17) * 5

plt.plot(x, y1, marker='*', color='lavender', label='Laven')
plt.plot(x, y2, marker='p', color='gray', label='Gray')
plt.plot(x, y3, marker='s', color='springgreen', label='Spri')

plt.legend() # label로 지정된 이름을 모아 범례 표시
plt.show()

📝 Matplotlib 핵심 요약

1. 그리기 (Draw)
  • plt.plot() : 선 그래프
  • plt.bar() : 막대 그래프
  • plt.scatter() : 산점도
  • plt.pie() : 원형 차트
2. 설정하기 (Set)
  • plt.title() : 제목
  • plt.xlabel() : X축 이름
  • plt.ylabel() : Y축 이름
  • plt.axis() : 축 범위
3. 꾸미기 (Style)
  • color : 색상
  • marker : 마커 모양
  • label + plt.legend() : 범례 표시

Module 4. 그래프 포함 엑셀 보고서 자동화

📉 엑셀로 내보내기 (to_excel)

파이썬을 이용해 열심히 그룹화하고 분석해 낸 결과를 다시 팀장님께 보고할 수 있도록 excel 파일로 내보내는 자동화의 핵심 기술입니다.

그래프 이미지를 엑셀 시트에 삽입하려면 xlsxwriter 같은 추가 엔진이 필요합니다.

BASH 
pip install openpyxl xlsxwriter
PYTHON 
import pandas as pd
import matplotlib.pyplot as plt

# 1. 실제 엑셀 파일 로드 및 지역별 매출액 그룹화 분석
df = pd.read_excel('sales_data.xlsx')
summary_df = df.groupby('지역')['매출액'].sum().reset_index()

# 2. 엑셀 워크북(문서) 객체 생성
writer = pd.ExcelWriter('report_with_chart.xlsx', engine='xlsxwriter')

# 3. 요약된 데이터를 '보고서'라는 시트 이름으로 저장
summary_df.to_excel(writer, sheet_name='보고서', index=False)

# 4. 분석된 데이터로 막대 그래프 생성 후 이미지 파일로 잠깐 저장
plt.bar(summary_df['지역'], summary_df['매출액'], color='#38bdf8')
plt.title('지역별 총매출')
plt.savefig('chart_temp.png', bbox_inches='tight') # 이미지 파일 출력

# 4. 저장한 이미지 파일을 엑셀 시트 E2 셀 위치에 삽입
worksheet = writer.sheets['보고서']
worksheet.insert_image('E2', 'chart_temp.png')

# 5. 엑셀 저장 완료 후 임시 이미지는 삭제할 수 있습니다.
writer.close()
print("🎉 엑셀 자동화 보고서가 완성되었습니다!")

🎯 실전 학습 포인트 요약

  • 파이썬에서는 pandas 라이브러리를 통해 엑셀 데이터를 자유자재로 요리할 수 있습니다.
  • 순서는 항상 [파일 열기] -> [EDA 및 전처리(fillna/필터링)] -> [시각화 & 엑셀 저장] 흐름입니다.
  • xlsxwriter 엔진을 사용하면 파이썬에서 만든 이미지를 바로 엑셀 보고서에 구겨넣는 마법을 부릴 수 있습니다.
