Module 1. 판다스(Pandas) 분석 기초

🚀 1단계: 분석 환경 설정 및 데이터 로드

데이터 분석을 시작하기 위해 도구를 불러오고 엑셀 파일을 파이썬 세계로 가져오는 단계입니다.
엑셀 파일 데이터를 파이썬 안에서는 '데이터프레임(DataFrame)'이라고 부르는 2차원 표 형태로 다루게 됩니다.

💡 주요 라이브러리 (Library)
- pandas: 엑셀처럼 표 형태의 데이터를 가공하고 조작하는 핵심 도구입니다.
- matplotlib.pyplot: 분석한 데이터를 알아보기 쉽게 그래프로 그려줍니다.

환경 세팅 (터미널에서 설치)

Visual Studio Code 터미널에서 아래 명령어를 사용해 필수 도구를 설치합니다.

BASH 
pip install pandas matplotlib

🛠️ 2단계: 버전을 확인하고 테스트 해보기

모든 설치가 끝났다면 간단한 가상의 데이터를 만들어서 표(DataFrame)로 변경하고 시각화까지 테스트해 봅시다.

PYTHON 
import pandas as pd
import matplotlib.pyplot as plt

# 1. 파이썬 딕셔너리를 활용해 간단한 데이터 생성
data = {
    '연도': [2023, 2024, 2025, 2026],
    '수치': [10, 25, 45, 30]
}

# 2. 데이터를 마법의 표인 데이터프레임으로 변환
df = pd.DataFrame(data)
print(df)

# 3. 데이터의 '연도'와 '수치'를 이용해 그래프 그리기
plt.plot(df['연도'], df['수치'], marker='o')
plt.title('Install Test')
plt.show()

Module 2. 전처리와 핵심 그래프 시각화

🔍 1단계: 안전하게 데이터 불러오기 (EDA)

실제 엑셀 데이터를 불러오고 제대로 들어왔는지 요약 정보를 탐색(EDA)합니다.

⚠️ 주의사항: 파일을 열 때 파일이 없으면 에러가 나서 프로그램이 다운됩니다. os.path.exists()를 사용해 파일이 있는지 사전에 검사하는 것도 좋은 실무 습관입니다.
PYTHON 
import pandas as pd
import os

file_name = 'sales_data.xlsx'

if os.path.exists(file_name):
    # 엑셀 파일 읽기
    df = pd.read_excel(file_name)
    print(f"✅ 로드 성공! 크기: {df.shape}")
    
    # 탐색적 데이터 분석 (EDA) 기초
    print(df.info())      # 데이터 타입, 비어있는 값 유무 확인
    print(df.describe())  # 기초 통계 요약
    print(df.head())      # 처음 5줄 미리보기
else:
    print("❌ 에러: 파일을 찾을 수 없습니다.")

🧹 2단계: 데이터 전처리 (Cleansing & Filtering)

현업의 데이터는 깨끗하지 않습니다. 빈 값을 채워주고 필요한 데이터만 쏙쏙 뽑아내야 합니다.

PYTHON 
# 1. 엑셀의 빈 칸(결측치, NaN)을 0으로 채우기
df = df.fillna(0)

# 2. 다중 조건 필터링: 수량이 5개 이상이고 지역이 '서울'인 데이터만 타겟팅
# 팁: 판다스에서 `그리고(AND)`는 `&` 기호를 사용합니다! (파이썬 기본의 and가 아님에 주의)
target_df = df[(df['수량'] >= 5) & (df['지역'] == '서울')]

# 3. 데이터 프레임에서 '품목'과 '매출액' 열(Column)만 떼어내기
subset = df[['품목', '매출액']]
print(subset)

Module 3. 그래프 포함 엑셀 보고서 자동화

📉 엑셀로 내보내기 (to_excel)

파이썬을 이용해 열심히 그룹화하고 분석해 낸 결과를 다시 팀장님께 보고할 수 있도록 excel 파일로 내보내는 자동화의 핵심 기술입니다.

그래프 이미지를 엑셀 시트에 삽입하려면 xlsxwriter 같은 추가 엔진이 필요합니다.

BASH 
pip install openpyxl xlsxwriter
PYTHON 
import pandas as pd
import matplotlib.pyplot as plt

# 가상의 지역별 매출 통계 데이터 시각화
summary_df = pd.DataFrame({'지역': ['서울', '부산'], '총매출': [5000, 3000]})

# 1. 엑셀 워크북(문서) 객체 생성
writer = pd.ExcelWriter('report_with_chart.xlsx', engine='xlsxwriter')

# 2. 요약된 데이터를 '보고서'라는 시트 이름으로 저장
summary_df.to_excel(writer, sheet_name='보고서', index=False)

# 3. 막대 그래프 생성 후 이미지 파일로 로컬에 잠깐 저장
plt.bar(summary_df['지역'], summary_df['총매출'], color=['#38bdf8', '#10b981'])
plt.title('지역별 총매출')
plt.savefig('chart_temp.png', bbox_inches='tight') # 이미지 파일 출력

# 4. 저장한 이미지 파일을 엑셀 시트 E2 셀 위치에 삽입
worksheet = writer.sheets['보고서']
worksheet.insert_image('E2', 'chart_temp.png')

# 5. 엑셀 저장 완료 후 임시 이미지는 삭제할 수 있습니다.
writer.close()
print("🎉 엑셀 자동화 보고서가 완성되었습니다!")

🎯 실전 학습 포인트 요약

  • 파이썬에서는 pandas 라이브러리를 통해 엑셀 데이터를 자유자재로 요리할 수 있습니다.
  • 순서는 항상 [파일 열기] -> [EDA 및 전처리(fillna/필터링)] -> [시각화 & 엑셀 저장] 흐름입니다.
  • xlsxwriter 엔진을 사용하면 파이썬에서 만든 이미지를 바로 엑셀 보고서에 구겨넣는 마법을 부릴 수 있습니다.
